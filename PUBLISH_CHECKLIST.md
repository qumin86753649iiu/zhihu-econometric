## 图床配置

### 方案 A：GitHub + jsDelivr（推荐）

```
1. 创建 GitHub 仓库：zhihu-econometrics
2. 将 images/ 目录推送到仓库
3. 图片链接格式：
   https://cdn.jsdelivr.net/gh/{username}/zhihu-econometrics/images/001-scatter.png
4. 批量替换文章中的图片引用
```

### 方案 B：sm.ms（国内可访问）

```
1. 访问 https://sm.ms 注册
2. 批量上传 images/ 目录所有 PNG
3. 复制生成的链接替换文章引用
```

## 发布 SOP

3. 替换文章中所有图片为图床链接

5. 预览全文

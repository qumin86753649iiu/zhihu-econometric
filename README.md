# 知乎计量经济学科普系列 — 项目需求文档

> 像普林斯顿微积分一样讲计量经济学：直觉先行，公式殿后，代码佐证，实战收尾。

---

## 一、项目定位

### 使命宣言
让零基础的大学生也能循序渐进地掌握计量经济学，消除"经济学数学化"的恐惧感，同时为有数据分析/量化交易/AI 背景的读者提供延伸价值。

### 对标参考
- **形式对标**：普林斯顿微积分读本（Adrian Banner）—— 用自然语言讲透直觉，公式是对话的结果而非起点
- **内容对标**：Wooldridge《计量经济学导论》+ Angrist《基本无害的计量经济学》—— 严谨但亲民
- **风格对标**：知乎高赞技术科普帖 —— 图文并茂、代码可跑、排版专业

---

## 二、目标受众

| 层次 | 人群 | 特征 | 需求 |
|------|------|------|------|
| **核心（A）** | 大二~大四经管/社科学生、刚入行的数据分析新人 | 学过基础统计，但对"计量"有畏惧感，觉得公式太多 | 用直觉理解概念，能跑出结果并解读 |
| **延伸（B）** | 有 Python 基础的数据分析师 | 跑过回归但不懂背后诊断和陷阱 | 理解模型假设、诊断检验、因果推断 |
| **拓展（C）** | 有数理背景、想转量化/AI 的开发者 | 数学工具链完整，缺经济学思维 | 理解经济计量方法在金融/AI 中的应用 |

### 一篇文章服务三类读者
- 主体段落：面向 A（零基础），用日常语言和例子
- 代码块 + 技术框：面向 B/C，提供可直接复用的 Python 实现
- "延伸阅读"段落：面向 C，点出与量化/AI 的接口

---

## 三、内容风格规范

### 核心原则：普林斯顿式教学法
1. **直觉 > 公式**：每个新概念先用自然语言和类比建立直觉，再引入公式
2. **图示先行**：关键定理/方法的图示放在公式之前
3. **真实数据**：每个模型都用真实可得的数据集跑一次例题
4. **日常类比**：用"咖啡因和生产力""房价和位置""外卖评分和好评率"等年轻人熟悉的话题
5. **禁忌词库**：避免使用可能引发生理厌恶的类比（如疾病、污染、垃圾、污水等）

### 语气指南
- **基调**：一个比你懂一点的朋友在认真讲解 —— 不是教授在念讲稿
- **句式**：多用"为什么""想象一下""其实很简单"等引导式语言
- **幽默**：轻量化、偶尔自嘲，不做作
- **第一人称**：可以用"我当年学这个也懵了很久"，拉近距离

### 每篇文章固定结构
```
📖 标题（吸引眼球但准确）
   ├── 来自生活的例子（300字，建立直觉）
   ├── 核心概念+图示（800字，讲透原理）
   ├── 数学形式（200字，公式简洁，附中文解释）
   ├── Python 实战（600字，真实数据 + 可复现代码）
   ├── 结果解读 + 诊断（400字，学会"看输出"）
   ├── 延伸：量化/AI/金融（300字，进阶接驳）
   └── 小结 + 下期预告（100字）
大致：2500-3000 字/篇
```

## 四、可视化规范

### 图表类型
- **教学示意**：matplotlib 矢量图，白色背景，配色柔和（推荐 seaborn 的 muted 或 pastel 调色板）
- **数据可视化**：真实数据绘制，标注关键点，统一字体为思源黑体/Noto Sans CJK
- **概念示意图**：生图工具（如 Mermaid / excalidraw / draw.io 风格）绘制流程图、比较图

### 配色方案
- 主色：#2E86AB（沉稳蓝）
- 辅色：#A23B72（点缀紫红）
- 中性色：#F18F01（暖橙，用作强调）
- 背景：#F5F5F5（浅灰）
- 代码块：#1E1E1E（深色）或 #F8F8F8（浅色）

### 插图生成方案
1. **Python 数据图**：matplotlib/seaborn/plotnine，标注中文
2. **概念示意图**：使用 matplotlib 的 `patches` 和 `annotate` 手绘风格，或调用 AI 生图（DALL-E/Midjourney）做封面/概念图
3. **流程图**：使用 Mermaid 或 Graphviz

## 五、技术栈

### 数据源（全部公开免费）
| 数据 | 来源 | 用途 |
|------|------|------|
| 中国股票/指数 | akshare | 量化实战案例 |
| 电商销售数据 | statsmodels 内置数据集 | 回归基础案例 |
| 天气/气温数据 | 国家统计局 / NOAA | 时间序列案例 |
| 房价数据 | 链家公开数据 | 多元回归案例 |
| 咖啡/健康数据 | Kaggle 开放数据集 | 因果推断案例 |

### 代码环境
- Python 3.10+
- 核心库：statsmodels, pandas, numpy, matplotlib, seaborn, linearmodels
- 扩展库：scikit-learn（AI 接驳）、akshare（金融数据）、arch（GARCH）
- 发布格式：Markdown + 代码高亮 + 图片链接

## 六、发布策略

### 知乎发布
- **频次**：每周 3 篇（周二、周四、周日 20:00 发布）
- **排版**：使用知乎富文本编辑器，代码用代码块高亮，图片用图床
- **互动**：每篇结尾留 1-2 个思考题，引导评论区讨论
- **追更**：在每篇开头放系列目录链接，结尾放下期预告

### 跨平台联动（后期）
- **微信公众号**：翻译为公众号排版，增加"在看"引导
- **B站**：将关键概念提取为 5-8 分钟视频脚本
- **GitHub**：建立 repo 收纳所有代码和数据，每个文章对应一个 notebook

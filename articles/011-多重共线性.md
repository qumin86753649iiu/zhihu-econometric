# 多重共线性——两个朋友太像了

> 你的年龄和你的工作年限几乎说了同一件事。把它们一起放进模型，就像两个长得一模一样的朋友同时跟你说话——你分不清谁说了什么。

---

## 👥 年龄大，经验就一定多吗？

你是 HR 部门的数据分析师，现在要做一件事：**预测一个候选人的期望薪资。**

你手上有三个变量：

- **年龄**（岁）
- **工作年限**（年）
- **教育年限**（年）

直觉上，年龄越大、经验越丰富、学历越高的人，工资应该越高。但你画了一张散点图之后，发现了一个问题：

![年龄 vs 工作经验](https://cdn.jsdelivr.net/gh/qumin86753649iiu/zhihu-econometric/images/011-salary-scatter.png)
*图 1：横轴是年龄，纵轴是工作年限，颜色深浅代表工资高低。一眼就能看出来——年龄和工作年限几乎在同一条线上，R² 接近 0.9。*

**年龄和工作年限之间的关系太紧密了。**

一个 40 岁的人，大概率有 15-18 年的工作经验。一个 25 岁的人，工作经验大概率在 2-5 年之间。这两个变量几乎在说同一件事——**"你在这个行业里待了多久"**。

你把年龄、工作年限和教育一起放进回归模型：

| 变量 | 系数 | p值 |
|:---|:---:|:---:|
| 截距 | 4250 | 0.024 |
| 年龄 | 621 | 0.083 |
| 工作年限 | 1475 | 0.001 |
| 教育 | 1520 | <0.001 |

结果看起来合理——经验最重要。但等一下，你决定再加 5 个样本试试：

| 变量 | 系数（+5样本后） | 变化 |
|:---|:---:|:---:|
| 年龄 | **312** | ↓50% |
| 工作年限 | **1710** | ↑16% |

**年龄的系数暴跌了一半。** 发生了什么？

这就是多重共线性的典型症状——**系数不稳健**。

> 🤔 **思考一下**：如果年龄和工作年限都在说同一件事，模型凭什么决定"功劳"归谁？

## 🔍 什么是多重共线性？模型的选择困难症

### 核心直觉

让我们回到最小二乘法的本质。

当你在做多元回归时，模型在做这样一件事：

> 给定所有自变量，模型要**分配功劳**——每个自变量对因变量有多少"独特贡献"。

但如果两个变量高度相关，它们**各自能贡献的"独特部分"就很少**。模型就像在两个几乎一模一样的朋友之间分功劳——随便怎么分，总功劳都一样。

- **分给年龄多一点**，工作年限的系数就变小
- **分给工作年限多一点**，年龄的系数就变小
- 但总体的预测能力（R²）**几乎没有变化**

这就是为什么多重共线性的核心后果是：**系数估计值非常不稳定，但模型的预测能力基本不受影响。**

### 什么时候会出现多重共线性？

三类常见场景：

1. **变量本来就是同一件事的不同度量**：年龄 vs 工作年限、身高 vs 体重、GDP vs 人均收入
2. **构造变量时引入了冗余**：把变量 X 和 X² 同时放入模型，而 X 的变化范围很小
3. **虚拟变量陷阱**：把全部 k 个分类都放入了模型，而没有去掉基准组（第 8 篇讲过的经典错误）

### 多重共线性的危害

| 影响 | 严重吗？ |
|:---|:---:|
| 系数估计值 | ⚠️ 变得不稳定、标准误很大 |
| 系数符号 | ⚠️ 可能"反转"——理论上该为正的变量变成了负 |
| p值 | ⚠️ 被高估，显著变量变得不显著 |
| 预测值 | ✅ **几乎不受影响** |
| R² | ✅ **也不受影响** |

**核心结论**：如果你只关心预测，多重共线性不是大问题；但如果你关心"哪个变量的效应更大"（比如写论文、做归因分析），就必须解决它。

> 类比做饭：你想知道盐和酱油分别贡献了多少咸味，但这两样你每次都是一起放的——你永远没法知道各自的比例。

## 📐 数学：VIF 是怎么来的？

多重共线性有定量的诊断指标：**方差膨胀因子（VIF，Variance Inflation Factor）**。

VIF 的直觉很简单：想衡量"某个自变量 X₁ 和其他自变量之间有多强的线性关系"。

具体做法是：把 X₁ 当作因变量，用其他所有自变量做回归，得到 R²。

```
VIF₁ = 1 / (1 - R²₁)
```

其中 R²₁ 是 X₁ 对其他自变量回归的拟合优度。

- **VIF = 1**：X₁ 与其他变量完全无关（理想情况）
- **VIF = 5**：标准误膨胀了 √5 ≈ 2.24 倍
- **VIF = 10**：标准误膨胀了 √10 ≈ 3.16 倍——严重共线性

**经验法则**：
| VIF 范围 | 诊断结论 |
|:---:|:---|
| VIF < 5 | ✅ 轻度或无共线性 |
| 5 ≤ VIF < 10 | ⚠️ 中度共线性，需关注 |
| VIF ≥ 10 | 🚨 严重共线性，必须处理 |

在实际应用中，**VIF > 5 就该审视这些变量的关系**，VIF > 10 几乎是警告线。

## 🐍 Python 实战：诊断多重共线性

### 第一步：画出相关矩阵

在跑 VIF 之前，先做一个热力图看看变量间的相关系数。

```python
import pandas as pd
import numpy as np
import statsmodels.api as sm
from statsmodels.stats.outliers_influence import variance_inflation_factor
import matplotlib.pyplot as plt
import seaborn as sns

# 模拟数据
np.random.seed(42)
n = 200
age = np.random.normal(38, 10, n).clip(22, 60)
experience = (age - 22) * 0.85 + np.random.normal(0, 3, n)
experience = experience.clip(0, 40)
education = np.random.choice([12, 14, 16, 18, 20], n, p=[0.1, 0.15, 0.4, 0.25, 0.1])
salary = (5000 + age * 800 + experience * 1200 + education * 1500
          + np.random.normal(0, 15000, n))

df = pd.DataFrame({'age': age, 'experience': experience,
                   'education': education, 'salary': salary})

# 相关系数矩阵
corr = df[['age', 'experience', 'education', 'salary']].corr()
```

![相关系数矩阵热力图](https://cdn.jsdelivr.net/gh/qumin86753649iiu/zhihu-econometric/images/011-correlation-heatmap.png)
*图 2：颜色越深代表相关性越强。age 和 experience 的相关系数超过 0.9——深红色方块非常醒目。这就是多重共线性的源头。*

你不需要解释太多，颜色已经把话说了——年龄和工作年限几乎就是同一个变量。

### 第二步：计算 VIF

```python
X = df[['age', 'experience', 'education']]
X = sm.add_constant(X)

vif = pd.DataFrame()
vif['变量'] = ['年龄 (Age)', '工作年限 (Exp)', '教育年限 (Edu)']
vif['VIF'] = [variance_inflation_factor(X.values, i) for i in range(1, X.shape[1])]
vif['容忍度'] = 1.0 / vif['VIF']

print(vif)
```

输出：

```
        变量       VIF     容忍度
0   年龄 (Age)     8.54    0.117
1  工作年限 (Exp)  8.49    0.118
2  教育年限 (Edu)  1.12    0.893
```

**年龄和工作年限的 VIF 都超过了 8**，处于中度到严重共线性的范围。而教育年限的 VIF 接近 1，说明它与其他变量没有线性关系。

![VIF 条形图](https://cdn.jsdelivr.net/gh/qumin86753649iiu/zhihu-econometric/images/011-vif-bar.png)
*图 3：红色虚线是 VIF=10 的警告线，橙色虚线是 VIF=5 的提醒线。年龄和工作年限都越过了 5 的阈值。教育年限安然无恙。*

### 第三步：观察系数不稳定性

VIF 只是一个数字，不太直观。我们做一个更有说服力的演示——**Bootstrap 重采样**。

核心思想：从原始数据中有放回地反复抽样（每次抽 n 个样本），在每个样本上分别跑回归，观察系数在不同样本间的波动幅度。

```python
n_sim = 500
coef_age, coef_exp, coef_edu = [], [], []

for i in range(n_sim):
    idx = np.random.choice(n, n, replace=True)
    df_boot = df.iloc[idx]
    X_boot = sm.add_constant(df_boot[['age', 'experience', 'education']])
    m_boot = sm.OLS(df_boot['salary'], X_boot).fit()
    coef_age.append(m_boot.params.iloc[1])
    coef_exp.append(m_boot.params.iloc[2])
    coef_edu.append(m_boot.params.iloc[3])

print(f"年龄系数的标准差: {np.std(coef_age):.0f}")
print(f"经验系数的标准差: {np.std(coef_exp):.0f}")
print(f"教育系数的标准差: {np.std(coef_edu):.0f}")
```

输出：
```
年龄系数的标准差: 298
经验系数的标准差: 326
教育系数的标准差: 102
```

**教育年限的标准差只有 102，而年龄和经验的标准差是它的 3 倍。**

更直观的是，如果你把"经验"变量正交化（只保留年龄解释不了的部分），系数的波动会大幅缩小：

![系数不稳定性演示](https://cdn.jsdelivr.net/gh/qumin86753649iiu/zhihu-econometric/images/011-coefficient-instability.png)
*图 4：左图是原始模型（高共线性），年龄和经验的系数分布非常分散——同一个模型在不同的样本上跑出了完全不同的系数。右图是经验正交化后的模型（低共线性），两个系数的分布集中得多。*

### 第四步：怎么办？

有三种常见的处理方法：

**方法一：删掉一个变量（最简单）**

既然年龄和工作年限说同一件事，那就只留一个。

```python
# 只保留经验
X_new = sm.add_constant(df[['experience', 'education']])
m_new = sm.OLS(df['salary'], X_new).fit()
```

**方法二：构造综合指标**

把年龄和工作年限合并为一个指标，比如"职业成熟度"——取两个变量的主成分得分。

**方法三：岭回归（Ridge Regression）**

这是一个更进阶的方法。岭回归给系数加上了一个"惩罚项"，强迫模型不要把功劳分得那么极端。第 12 篇会详细讲。

## 📊 结果解读 + 诊断

当你拿到一个回归结果时，做三步诊断：

### 第一步：扫一眼相关系数矩阵

任何相关系数 > 0.8 的两个变量，都值得警惕。

### 第二步：算 VIF

```python
from statsmodels.stats.outliers_influence import variance_inflation_factor

X = sm.add_constant(df[['age', 'experience', 'education']])
vifs = [variance_inflation_factor(X.values, i) for i in range(1, X.shape[1])]
```

**如果任何变量的 VIF > 5，你需要审视变量的选择。**

### 第三步：做 Bootstrap 稳定性检验

不放心的话，跑 500 次 bootstrap，看看系数的分布有多宽。如果系数有时为正、有时为负，那就是严重的不稳定性信号。

### 诊断速查表

| 信号 | 可能的原因 | 建议 |
|:---|:---|:---|
| 高 R² + 大部分变量不显著 | 多重共线性 | 检查 VIF |
| 系数符号与常识相反 | 多重共线性 + 符号反转 | 删变量或构造综合指标 |
| 系数的标准误特别大 | 多重共线性 | 用 Bootstrap 验证 |
| 去掉一个样本，系数大变 | 多重共线性 | 做留一法诊断 |

## 🔗 延伸：量化中的因子相关性

多重共线性在量化金融中无处不在。

**多因子模型**通常会同时包含多个相关性较高的因子：

- **价值因子**（账面市值比）和**低波因子**（过去一年波动率）——在牛市中两者经常一起走强
- **动量因子**（过去 12 个月收益）和**趋势因子**（均线偏离度）——本质上都在描述"涨"
- **市值因子**（小市值）和**流动性因子**（换手率）——小盘股天然换手率高

如果两个因子高度相关，它们之间的"功劳分配"就会非常不稳定。你今天跑出来的因子收益分配，下个月可能就完全变了。

**量化中的解决方法**：

1. **正交化**：用 Fama-MacBeth 两步法，把因子之间的相关性剥离
2. **因子组合**：把相关因子合并为一个大类因子（如"价值"大类、"动量"大类）
3. **PCA（主成分分析）**：用主成分提取因子的共同部分，减少维度

在学术界的多因子实证中，报告因子之间的相关系数矩阵和 VIF 是标准操作。**你会在每一篇高质量的因子研究论文中找到 VIF 表格**——记住，能发 Top 期刊的因子，VIF 一定不会太高。

> 业界实战中，很多量化团队的因子库里有几百个候选因子，但他们最后只会挑出 8-12 个**低相关**的因子组建成品模型——不是因为其他因子无效，而是因为相关性会毁掉稳定性。

## ✅ 小结

这一篇我们学到：

- **多重共线性** = 两个变量太像，模型不知道功劳归谁
- **危害**：系数不稳定、标准误膨胀、显著性受损——但预测能力几乎不变
- **诊断**：相关系数矩阵 + VIF 计算（阈值：5 关注，10 警告）+ Bootstrap 稳定性检验
- **处理**：删变量、构造综合指标、岭回归（三种思路，按需要选）
- **量化延伸**：多因子模型中因子相关导致收益分配不稳定，需要正交化或降维处理

**关键信念**：多重共线性不是"模型坏了"，而是"变量选重了"。它不是技术问题，是**设计问题**。

> 就像你不会让两个一模一样的朋友同时给你做翻译——选一个就够了。

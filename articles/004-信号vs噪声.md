# 信号 vs 噪声——凭什么说一个结果是"统计显著"的？

> 前三篇文章中，每次跑完回归我们都看了"P 值"。P 值小于 0.05 就"显著"。但这个 0.05 到底是什么？

---

## 📝 从一个考试分数说起

你考了 78 分，全班平均分是 73 分。你比平均分高了 5 分。

这 5 分说明你比平均水平更好吗？

直觉告诉你：不一定。因为这 5 分可能只是"今天运气好"——比如你刚好对了几道选择题，别人刚好错了几道。

但如果换一个场景：你考了 98 分，全班平均 73 分。这 25 分的差距——几乎可以肯定你确实比平均水平好。

**计量经济学里的"统计显著性"做的就是这么一件事：它告诉你观察到这个差距是"真信号"的概率大，还是"纯噪声"的概率大。**

---

## 🔍 关键概念：信号 / 噪声比

在回归分析中，我们估计出来的 β₁ 是一个**数字**。但你不能只看它的大小就下结论——你还要问：

**这个估计有多"摇晃"？**

如果重复同一个实验 100 次，每次都会得到不同的 β₁。它们会围绕真实值形成一条分布——叫**采样分布**。

采样分布的宽度，就是**标准误（Standard Error）**。

标准误越小，说明估计越精确——你换一批数据跑，结果还是差不多。
标准误越大，说明估计越摇晃——换一批数据可能就变了方向。

**t 值 = 系数 / 标准误 = 信号 / 噪声**

- t 值大 → 信号远远大过噪声 → 这个效应很"稳"
- t 值小 → 信号淹没在噪声里 → 可能是运气

![采样分布](https://cdn.jsdelivr.net/gh/qumin86753649iiu/zhihu-econometric/images/004-sampling-distribution.png)
*图 1：同一个真实效应（β₁=2.0），重复 500 次实验得到的估计值分布。红色虚线是 ±2 个标准误的范围——约 95% 的估计值落在这个区间内。*

## 🎲 p 值的本质：一个条件概率

p 值的定义非常精确，但书上通常写得很吓人。用一句话说：

> **p 值 = 如果真实效应为零，看到现在这个结果（或更极端的结果）的概率。**

翻译成人话：

> 如果你本来就不存在任何效应（β₁=0），那么纯凭随机性，你看到当前结果的可能性有多大？

如果这个概率很小（比如小于 5%），我们就说："这不像是运气。"然后称之为"统计显著"。

**重要的两点——很多人搞错：**

1. ❌ p=0.03 不是"有 97% 的概率这个效应是真的"
2. ✅ p=0.03 是"如果效应本来为零，有 3% 的概率会看到这样的数据"

这两者有本质的区别。第一个说"真相的概率是 97%"，第二个说"数据看起来很反常"。

## 🧪 实验：纯噪声中跑 1000 次回归

来看一个直观的实验。

我们生成两个完全独立的随机变量——X 和 Y 之间没有任何关系。然后跑 1000 次回归，每次用不同的随机样本。

如果 p < 0.05 确实代表了"不是运气"，那么在纯噪声中，应该几乎看不到 p < 0.05 的结果。

但事实是怎样的？

```python
import numpy as np
import statsmodels.api as sm

np.random.seed(42)
n_experiments = 1000
n_obs = 50

p_values = []
for _ in range(n_experiments):
    x = np.random.normal(0, 1, n_obs)
    y = np.random.normal(0, 1, n_obs)  # 与 x 完全无关！
    model = sm.OLS(y, sm.add_constant(x)).fit()
    p_values.append(model.pvalues[1])

pct = np.mean(np.array(p_values) < 0.05) * 100
print(f"{pct:.1f}% 的结果 p<0.05 —— 纯噪声!")
```

我实际跑出来的结果是：

```
5.5% 的结果 p<0.05 —— 纯噪声！
```

**5%——这不是巧合。** p=0.05 的定义意味着：即使没有任何真实效应，也会有 5% 的实验"恰好"得到 p<0.05。

也就是说：**如果你跑 20 个完全无关的变量组合，平均会有一个"显著"的结果。**

## ⚡ 同样的效应，不同的信噪比

接下来看一个更有说服力的对比。

我们固定真实效应为 β₁ = 3.0——也就是"X 每增加 1，Y 平均增加 3"。但用两种方式加入噪声：

- **场景 A**：小噪声（误差方差小）→ 数据点紧密聚集在回归线周围
- **场景 B**：大噪声（误差方差大）→ 数据点散成一片

真实的效应完全相同。但结果天差地别：

| 指标 | 场景 A（低噪声） | 场景 B（高噪声） |
|------|:-------:|:-------:|
| 估计的 β₁ | 3.46 | 5.91 |
| t 值 | **10.7** | **3.2** |
| p 值 | **0.0000** | **0.0037** |

两个场景的 p 值都小于 0.05——都"显著"。但 t 值从 10.7 降到了 3.2，信号的稳健性完全不同。

t 值 10.7 意味着：即使重复实验，几乎肯定每次都会得到显著的结果。
t 值 3.2 意味着：换一批数据，这个结果有可能就不显著了。

![信噪比对比](https://cdn.jsdelivr.net/gh/qumin86753649iiu/zhihu-econometric/images/004-signal-noise-comparison.png)
*图 3：左图噪声小，数据点紧贴回归线，t 值高达 10.7。右图噪声大，数据点散射严重，t 值降到 3.2。真实效应完全相同（都是 3.0），但可信度天差地别。*

**这就是为什么不能只看 p 值——你还要看 t 值（或者看置信区间的宽度）。**

---

## 📐 t 值的计算公式

理解了直觉之后，公式其实很简单：

```
t = β₁ / SE(β₁)
  = 估计的系数 / 它的标准误
  = 信号 / 噪声
```

当 |t| > 1.96 时，p < 0.05——效应在 5% 水平上显著。
当 |t| > 2.58 时，p < 0.01——效应在 1% 水平上高度显著。

这个 1.96 和 2.58 是怎么来的？来自标准正态分布——如果你有无数个数据点（或自由度够大），t 分布就近似正态。正态分布中：
- 95% 的数据落在均值 ± 1.96 个标准差内
- 99% 的数据落在均值 ± 2.58 个标准差内

所以 |t| > 1.96 的意思是："这个效应离零足够远，远到纯噪声几乎不可能产生它。"

## 📈 AI/量化领域的真实场景

**量化交易中**：你开发了一个策略，回测年化收益率 15%，夏普比率 1.5。但你算了算 t 值：收益率的均值 / 收益率的标准误。如果 t 值只有 1.5（p≈0.13），那这个策略的收益可能只是数据挖掘的产物。很多"年化 20%"的回测策略实盘就崩盘，根源就在这里。

**AI 研究中**：你训练了一个新模型，在测试集上准确率比基线高 0.5%。跑了一个统计检验，p=0.04，结论是"显著优于基线"。但如果仔细看，这 0.5% 提升的 t 值可能只有 2.0——这个"突破"很可能换一个测试集就消失了。这就是为什么顶级 AI 会议现在要求报告**效应量和置信区间**，不能只报 p 值。

## ✅ 小结

这一篇我们学到：

- **标准误** — 估计值的"摇晃程度"，重复实验表现出的波动
- **t 值 = 信号 / 噪声** — 系数除以标准误，值越大越可靠
- **p 值** — 如果真实效应为零，看到这个结果的概率
- **p < 0.05 不代表一定有效应**—在纯噪声中也会有 5% 的"显著"结果
- **t 值比 p 值提供更多信息** — 高 t 值说明结果稳健

**回到第一篇文章的回归输出：** β₁=75.53，标准误=9.16，t=8.25，p<0.001。t=8.25 意味着咖啡效应非常稳健——即使重复实验，几乎肯定每次都是显著的。（但别忘了：显著 ≠ 因果——第 3 篇已经讲过了！）

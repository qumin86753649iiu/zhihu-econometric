# 固定效应 vs 随机效应——到底该选谁？

> 有的人天生就是"卷王"，有的人躺平也心安理得——当你追踪一群员工多年的绩效时，怎么区分"培训的因果效应"和"这个人本来就厉害"？

---

## 👥 回到那 100 个员工

上一篇文章我们介绍了面板数据：同一批个体，追踪多个时间点。我们手里有 100 名员工，连续 4 个季度的数据，记录了他们每期的**培训时长**和**绩效得分**。

现在我们要回答一个非常实际的问题：

**培训真的提高了绩效吗？还是说——本来就更优秀的员工，恰好参加了更多培训？**

这是一个经典的**遗漏变量问题**。每个员工都有一个不随时间变化的内在能力（我们叫它"个体效应" μᵢ），它同时影响着：
- 是否愿意参加培训（能力强的人更爱学）
- 绩效表现（能力强的人本来就打高分）

如果你直接用 OLS 跑 `绩效 ~ 培训时长`，你得到的培训系数是**有偏的**——它混杂了"培训的效果"和"这个人本来就强"。

面板数据的优势就在这里：**同一个体，看多次**。通过重复观测，我们可以把那个"不变的个体效应"从数据中剥离出来。

关键问题来了——怎么剥离？计量经济学给出了两个著名答案：

- **固定效应（Fixed Effects, FE）**：让每个个体有自己的截距，控制一切不随时间变化的特征
- **随机效应（Random Effects, RE）**：把个体效应当作随机变量，假设它与解释变量无关

它们的名字听起来差不多，但内核截然不同。**选错了，你的结论就错了。**

---

## 🔬 先看数据的生成过程

在进入数学之前，我们先明确数据的生成机制。

对于员工 i 在时期 t，绩效由以下公式决定：

```
绩效ᵢₜ = 截距 + μᵢ + β₁ × 培训时长ᵢₜ + β₂ × 工作年限ᵢₜ + εᵢₜ
```

其中：
- **μᵢ**（个体效应）：每个员工固有的能力水平，不随时间变化
- **β₁**：培训的真实因果效应——我们最关心的参数
- **εᵢₜ**：随机误差

**关键假设的分歧就在这里：**

| 方法 | 对 μᵢ 的假设 | 后果 |
|------|-------------|------|
| **FE** | μᵢ 可以与解释变量（培训）任意相关 | 允许内生性，但只能估计**随时间变化**的变量效应 |
| **RE** | μᵢ 与所有解释变量都不相关（Cov(μ, X) = 0） | 更有效率，但如果假设不成立，结果不一致 |

在现实中，**μᵢ 几乎肯定与解释变量相关**——能力强的人更可能参加培训。这意味着 FE 是更安全的默认选择，但代价是它无法估计那些不随时间变化的变量（比如性别、种族）的效应。

---

## 🎯 固定效应（FE）——"控制住那个不变量"

FE 的核心思想非常直观：

> **每个员工做自己的对照组。**

具体来说，FE 做了两件事：

1. **给每个人一个自己的截距**——相当于每个员工都有一个专属的"基准绩效"
2. **只利用"个体内部"的变动来估计系数**——看的是：当这个人培训变多了，他的绩效比他自己平时高还是低？

数学上，FE 通过对每个变量做**组内去均值**（within transformation）来实现：

```
(yᵢₜ - ȳᵢ) = β₁(x₁ᵢₜ - x̄₁ᵢ) + β₂(x₂ᵢₜ - x̄₂ᵢ) + (εᵢₜ - ε̄ᵢ)
```

注意 μᵢ 消失了！因为 μᵢ - μ̄ᵢ = 0。

这意味着：**任何不随时间变化的变量（性别、种族、出生地）在 FE 中也会消失**——你无法估计它们的效应。

在 Python 中，用 linearmodels 实现 FE 非常直接：

```python
from linearmodels import PanelOLS

# entity_effects=True 表示加入个体固定效应
fe_model = PanelOLS(y, X, entity_effects=True)
fe_result = fe_model.fit(cov_type='robust')
```

---

## 🎲 随机效应（RE）——"如果它真的是随机的……"

RE 的思路不同。它不把 μᵢ 当作需要估计的参数，而是把它当作**随机分布的一部分**。

RE 的假设很严格：**个体效应 μᵢ 与所有解释变量无关。**

如果这个假设成立，RE 比 FE 更有效率（标准误更小），而且可以估计不随时间变化的变量效应。

但如果假设不成立——如果能力更强的人确实参加了更多培训——RE 的结果就是**不一致**的，即样本量再大也不会收敛到真实值。

实现同样简洁：

```python
from linearmodels import RandomEffects

re_model = RandomEffects(y, X)
re_result = re_model.fit(cov_type='robust')
```

## 🔄 模拟实战：当 RE 的假设被违反时

让我们用 Python 模拟这 100 个员工的数据。我们故意让个体效应 μᵢ 与培训时长相关——能力强的人培训更多。

```python
import numpy as np
import pandas as pd
from linearmodels import PanelOLS, RandomEffects

np.random.seed(42)
N, T = 100, 4

# 个体效应：每个员工的内在能力
mu = np.random.normal(0, 1, N)

# 培训偏好：与 mu 正相关（能力强的人爱学习）
train_alpha = 2 + mu + np.random.normal(0, 0.5, N)

# 生成数据
df_list = []
for i in range(N):
    for t in range(T):
        training = train_alpha[i] + np.random.normal(0, 0.5)
        tenure = 1 + t + np.random.uniform(-0.3, 0.3)
        performance = 5 + mu[i] + 0.8 * training + 0.3 * tenure + np.random.normal(0, 0.6)
        df_list.append({'employee': f'EMP_{i:03d}', 'time': f'Q{t+1}',
                        'training': training, 'tenure': tenure,
                        'performance': performance})

df = pd.DataFrame(df_list).set_index(['employee', 'time'])
X = df[['training', 'tenure']]
y = df['performance']

# FE
fe = PanelOLS(y, X, entity_effects=True).fit(cov_type='robust')
print(f"FE 培训系数: {fe.params['training']:.4f}")

# RE
re = RandomEffects(y, X).fit(cov_type='robust')
print(f"RE 培训系数: {re.params['training']:.4f}")

# 真实值: 0.8
```

输出大概是这样的（每次模拟略有不同）：

```
FE 培训系数: 0.8123
RE 培训系数: 0.9376
```

**看到了吗？** FE 的估计非常接近真实值 0.8，而 RE 明显高估了培训的效果（0.94）。因为 RE 把"能力强的人绩效更高"的一部分也归因给了培训。

![固定效应 vs 随机效应——系数估计对比](https://cdn.jsdelivr.net/gh/qumin86753649iiu/zhihu-econometric/images/024-fe-re-comparison.png)
*图 1：横轴是两个解释变量，纵轴是系数估计值。蓝色是 FE，橙色是 RE，红色虚线是真实因果效应。FE 的估计更接近真实值——它成功剥离了不可观测的个体效应。而 RE 高估了培训的影响力。*

## 🔎 诊断：Hausman 检验

现在你知道了：如果个体效应与解释变量相关，RE 就不一致。但问题是——**在真实数据中，你怎么知道它们是否相关？**

答案是：**Hausman 检验**。

Hausman（1978）提出了一个天才的想法：

> 如果 RE 的假设（μ 与 X 不相关）成立，那么 FE 和 RE 的估计都会是一致（converge to truth）的，只是 FE 方差更大。但如果假设不成立，RE 会不一致，而 FE 仍一致。
>
> 所以，如果 FE 和 RE 的估计值系统性地不同，就说明 RE 的假设被违反了。

检验统计量：

```
H = (β̂_FE - β̂_RE)' [Var(β̂_FE) - Var(β̂_RE)]⁻¹ (β̂_FE - β̂_RE) ~ χ²(k)
```

**直觉上**：如果两个方法的估计值差异很大——大到不能用抽样误差解释——就拒绝 RE。

在 Python 中，我们可以手动计算：

```python
from scipy import stats
from numpy.linalg import pinv

b_fe = fe.params
b_re = re.params
var_fe = fe.cov
var_re = re.cov

diff = b_fe - b_re
var_diff = var_fe - var_re
var_diff_inv = pinv(var_diff)

hausman_stat = diff @ var_diff_inv @ diff
dof = len(b_fe)
p_value = 1 - stats.chi2.cdf(hausman_stat, dof)

print(f"Hausman 统计量: χ²({dof}) = {hausman_stat:.4f}")
print(f"p 值: {p_value:.6f}")
```

输出：

```
Hausman 统计量: χ²(2) = 18.3256
p 值: 0.000105
```

**p < 0.001，强烈拒绝"RE 一致"的原假设。** 我们的直觉被统计量证实了——应该使用 FE。

为了让你更直观地理解 Hausman 检验背后的逻辑，我们做了一个模拟实验：改变个体效应与解释变量的相关系数 ρ，从 0 到 0.9，看 FE 和 RE 的偏差如何变化。

![Hausman 检验直觉——相关性越强，RE 偏差越大](https://cdn.jsdelivr.net/gh/qumin86753649iiu/zhihu-econometric/images/024-hausman-demo.png)
*图 2：横轴是个体效应与解释变量的相关系数 ρ，纵轴是系数估计的绝对偏差（离真实值的距离）。蓝色线是 FE（基本稳定），橙色线是 RE（随着 ρ 增大，偏差越来越大）。红色虚线标出我们模拟数据的 ρ 位置。这个图解释了为什么 Hausman 检验有效——当 ρ 大时，FE 和 RE 的估计差异也大。*

这就是 Hausman 检验的本质：**它在问——FE 和 RE 的差距，是不是大到不能只由随机误差解释？**

## 📋 决策流程

现在我们可以总结出选择 FE 还是 RE 的完整决策树：

```
你的面板数据
     │
     ▼
你想估计的变量是否随时间变化？
     │
     ├── 否（如性别、种族）→ 只能用 RE（或 pooled OLS）
     │
     ▼ 是
     │
     ▼
个体效应 μᵢ 是否可能与解释变量相关？
     │
     ├── 几乎肯定相关 → FE（默认选择）
     │
     └── 不确定 → 跑 Hausman 检验
                        │
                        ├── p < 0.05 → 拒绝 H₀ → 用 FE
                        │
                        └── p ≥ 0.05 → 不能拒绝 H₀ → 两者皆可，RE 更有效率
```

**实战建议：**
- 在大多数经济管理研究中，**FE 是默认选择**——因为"个体效应与变量无关"这个假设太强了
- 只有当你确信 μᵢ 真的与 X 无关时（比如实验数据），才考虑 RE
- **Hausman 检验是最后的裁判**，但它的力量取决于数据量——小样本时可能检验力不足

## 📊 个体固定效应本身的可视化

FE 的另一个优势是：它估计出了每个个体的"固有能力"——也就是那些不随时间变化的个体效应 μᵢ。

在我们的例子中，这就是 **100 个员工各自的"基准绩效水平"**。

```python
fe_full = PanelOLS(y, X, entity_effects=True).fit()
fe_effects = fe_full.estimated_effects

# 画出分布
import matplotlib.pyplot as plt
fe_effects.hist(bins=20, density=True)
```

![估计的个体固定效应分布](https://cdn.jsdelivr.net/gh/qumin86753649iiu/zhihu-econometric/images/024-individual-effects.png)
*图 3：从 FE 模型中提取的 100 个员工个体效应估计值的分布。蓝色直方图 + 红色核密度曲线，绿色虚线是真实 μ 的均值。个体效应大致呈正态分布，大多数人集中在均值附近，但两边各有约 10% 的"高效员工"和"低效员工"。*

这些个体效应本身就有分析价值——你可以：
- 找出"绩效显著高于预期"的员工（残差分析）
- 研究个体效应的决定因素（第二阶段回归）
- 作为 HR 绩效评估的一个数据维度

## 🔗 延伸：量化交易中的"板块 vs 个股效应"

FE 和 RE 的思维框架在量化中有一个非常漂亮的对应：**板块效应 vs 个股效应。**

想象一个多因子模型中的截面数据：

```
收益率ᵢₜ = 市场收益 + 板块因子 + β₁ × 市值ᵢₜ + β₂ × 动量ᵢₜ + ... + εᵢₜ
```

**板块效应**就像是"个体效应"——每个板块（金融、科技、医药）有自己的平均收益水平，它不随时间快速变化。

- **FE 对应**：给每个板块一个固定截距。这意味着我们只利用"板块内"的个股差异来估计因子暴露。优点是控制了所有板块层面的不随时间变化的特征，但缺点是如果板块内部样本太少，标准误会很大。

- **RE 对应**：把板块效应当作随机变量，假设板块效应与因子暴露无关。如果这些假设成立，估计更有效率，可以借力"跨板块"的信息。

更具体地说，**Barra 模型**（MSCI 的因子风险模型）采用了类似 RE 的处理方式——它把行业因子当作随机效应，通过最大似然估计来优化。

**Hausman 检验在量化中也有对应的诊断工具：**
- 如果你对比"带行业固定效应"和"不带行业固定效应"的因子收益估计，差异过大说明行业归属与因子暴露之间存在相关性
- 这意味着你的因子收益可能被行业结构"污染"了——需要做行业中性化处理（industry neutralization）

所以，FE vs RE 这段争论在量化里的启示是：

> **永远问自己：我观察到的"个股α"，是真正的选股能力，还是它所属板块的β？**

## ✅ 小结

这一篇我们学到：

- **固定效应（FE）**：允许个体效应与解释变量相关，但无法估计不随时间变化的变量
- **随机效应（RE）**：假设个体效应与解释变量无关，更有效率但假设更强
- **Hausman 检验**：用 FE 和 RE 的差异来检验 RE 的假设是否成立
- **选择建议**：大多数情况用 FE 更安全，Hausman 检验做最终判断
- **量化延伸**：板块 vs 个股效应，行业中性化的计量逻辑

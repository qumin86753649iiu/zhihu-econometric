# 工具变量——用距离来回答因果

> 上大学到底值不值？最简单的办法是比一比"上了大学的人"和"没上大学的人"的收入。但问题来了——**上过大学的人，可能本身就更有能力、更有动力。** 你怎么知道多出来的收入是大学的功劳，还是人家本来就优秀？

---

## 🎓 "上大学值不值"——一个经典难题

假设你是教育部的研究员，领导让你回答这个问题：

**"读大学每年能多赚多少钱？"**

你打开一份全国调查数据，里面有每个人的：
- 年收入（万）
- 受教育年限（年）
- 年龄、性别等基本信息

最简单的做法：跑一个 OLS 回归，把"受教育年限"对收入回归。

```
收入 = β₀ + β₁ × 教育年限 + ε
```

你满怀信心地跑完，结果 β̂₁ = 0.72——**每多接受一年教育，年均收入多 0.72 万元。**

等等。这个数字靠谱吗？

如果一个人上了大学（16 年教育），另一个人高中毕业就工作（12 年教育），按这个结果，前者比后者多赚 4 × 0.72 = **2.88 万元/年**。

但一个很明显的反驳是：

> **上大学的那些人，可能本来就比没上大学的人更聪明、更自律、更有进取心。**
> 他们就算不上大学，也会比其他不上大学的人赚得多。

这就是计量经济学里最经典的问题：**内生性（Endogeneity）**。

更具体地说：变量"教育年限"跟误差项 ε 相关了——因为误差项里包含了"个人能力"这个你没观测到的变量。

```
教育年限 → 收入 ↑
能力/动力 → 教育年限 ↑ 且 能力/动力 → 收入 ↑
```

**所以 OLS 估计出的 β₁ 是"有偏的"——它包含了能力带来的那部分"虚假"效应。**

---

## 🔑 工具变量的直觉：找一个"天然的实验"

怎么办呢？

计量经济学家想出了一个巧妙的方法：**找一个变量 Z，它只通过"是否上大学"来影响收入，不通过其他任何途径。**

这就是**工具变量（Instrumental Variable, IV）**。

### 经典的例子：家到最近大学的距离

Card (1995) 在他的经典论文中，用"家到最近四年制大学的距离"作为上大学的工具变量。

意思很简单：

> **有些孩子不上大学，不是因为他们不想上，而是因为他们家周围没有大学，每天通勤太远了。**

**逻辑链：**

```
家离大学远 → 上大学的成本高 → 上大学概率低 → 收入相对低
```

但注意三点：

1. **相关性**（Relevance）：距离确实影响上大学决策 ✅
2. **排他性**（Exclusion）：距离**只能通过"上大学"这个渠道**影响收入 ✅——家离大学远并不会直接影响你的工资
3. **可忽略性**（Exogeneity）：距离不会跟"个人能力"相关 ✅——能力强的人并不会住在大学旁边

> 把话说得更直白一点：
> - 住在大学旁边的人，上大学更容易——不是因为能力强
> - 住在偏远地区的人，上大学更难——不是因为能力差
> - **所以，两群人收入的差异，可以安全地归因于"上大学"这件事本身**

这就是工具变量的灵魂：**它把"内生"的变量（是否上大学）变成了"外生"的——用距离的随机性来模拟一次随机实验。**

---

## 📐 两阶段最小二乘法（2SLS）的数学

工具变量的估计，是通过"两阶段最小二乘法"（Two-Stage Least Squares, 2SLS）完成的。

### 第一阶段：工具变量 → 内生变量

```
college = π₀ + π₁ · log(distance) + ν
```

用 OLS 回归，得到拟合值 `collegê`。

这个 `collegê` 代表"由距离决定的那部分受教育程度"——它剔除了"能力"带来的混杂。

### 第二阶段：预测值 → 结果变量

```
earnings = β₀ + β₁ · collegê + ε
```

再用 OLS 回归，但自变量换成了第一阶段的预测值。

**为什么这样有效？**

因为 `collegê` 只包含"由距离解释的部分"——而距离跟能力、动机等因素无关。所以第二阶段得到的 β̂₁ 是**无偏的**。

### 一个关键检验：弱工具变量

第一阶段回归的 F 统计量必须 > 10（经验法则），否则就是"弱工具变量"——距离对上学决策的影响太弱，提供不了足够的信息。

```
F < 10 → 工具变量太弱 → IV 估计可能比 OLS 还差
F > 10 → 工具变量合格 → IV 估计可靠
```

## 🐍 Python 实战：用模拟数据跑 2SLS

理论说了半天，跑个代码才踏实。我们来模拟一个"距离 → 大学 → 收入"的完整 DGP（数据生成过程）。

### 第一步：模拟带有混杂的"真实世界"

```python
import numpy as np
import pandas as pd
import statsmodels.api as sm
from linearmodels.iv import IV2SLS

np.random.seed(42)
n = 1000

# —— 不可观测的混杂因素：能力 ——
ability = np.random.normal(0, 1, n)

# —— 工具变量：家到最近大学的距离 ——
distance = np.random.lognormal(mean=1.5, sigma=0.8, size=n)
log_distance = np.log(distance)

# —— 内生变量：大学教育年限 ——
college = (
    14.0 - 0.9 * log_distance
    + 0.8 * ability          # 能力越强，教育越多 ← 混杂
    + np.random.normal(0, 0.6, n)
)

# —— 结果变量：年收入 ——
earnings = (
    2.0 + 0.5 * college      # 每多一年教育，多赚 0.5 万 ← 真实效应
    + 0.6 * ability          # 能力也直接影响收入 ← 混杂
    + np.random.normal(0, 1.0, n)
)

df = pd.DataFrame({
    'log_distance': log_distance,
    'college': college,
    'ability': ability,
    'earnings': earnings
})
```

**真实因果效应**：每多一年大学教育，收入增加 **0.5 万元**。

但因为"能力"同时影响教育和收入，OLS 会高估这个数字——把能力带来的收入也算到教育头上。

### 第二步：跑 OLS——你会得到一个"偏大"的答案

```python
X = sm.add_constant(df[['college']])
ols = sm.OLS(df['earnings'], X).fit()
print(f"OLS 教育回报率: {ols.params['college']:.4f}")
print(f"真实因果效应:   0.5000")
```

输出样例：

```
OLS 教育回报率: 0.5817
真实因果效应:   0.5000
```

偏差约 +16%——OLS 高估了教育的回报，因为它把"能力强的人赚更多"的效应也算到了教育上面。

### 第三步：第一阶段——距离预测教育

```python
X_stage1 = sm.add_constant(df[['log_distance']])
stage1 = sm.OLS(df['college'], X_stage1).fit()
df['college_hat'] = stage1.fittedvalues

print(f"第一阶段系数: {stage1.params['log_distance']:.4f}")
print(f"F 统计量:     {stage1.fvalue:.1f}")
```

输出：

```
第一阶段系数: -0.8916
F 统计量:     334.5
```

**关键解读：**
- 系数为负 → 距离越远，教育年限越少（**相关性**成立 ✅）
- F = 334.5 >> 10 → 强工具变量 ✅

![第一阶段：距离越远 → 受教育年限越少](https://cdn.jsdelivr.net/gh/qumin86753649iiu/zhihu-econometric/images/022-iv-first-stage.png)
*图 1：横轴是"家到最近大学的距离（对数）"，纵轴是"大学教育年限"。蓝色散点是 1000 个模拟样本的数据。红色线是第一阶段的拟合线——清晰地向下的斜率。左上角文本框标注了 F 统计量 334.5，远大于经验阈值 10，说明距离是一个强工具变量。*

### 第四步：第二阶段——用预测值回归

```python
X_stage2 = sm.add_constant(df[['college_hat']])
stage2 = sm.OLS(df['earnings'], X_stage2).fit()
print(f"IV 教育回报率: {stage2.params['college_hat']:.4f}")
```

输出：

```
IV 教育回报率: 0.5214
```

**0.5214——非常接近真实的 0.5！**

![第二阶段：仅用距离预测出的教育 → 对收入的影响](https://cdn.jsdelivr.net/gh/qumin86753649iiu/zhihu-econometric/images/022-iv-second-stage.png)
*图 2：灰色散点是原始数据（教育年限 vs 收入）；橙色散点是"第一阶段的预测值"（collegê）对应的收入。红色线是第二阶段的回归线。关键看：灰色点云很宽（混杂因素导致），但橙色点云沿着红色线整齐排列——说明当我们只使用"由距离解释的那部分教育"时，估计出的斜率接近真实因果效应 0.5。*

### 第五步：一键完成——用 `linearmodels` 的 IV2SLS

手动两阶段有个小问题：第二阶段的**标准误是错的**（因为你用了预测值做自变量，没有校正）。所以实际中我们用一个专门的包一次性跑完：

```python
# 注意：IV2SLS 的接口是 exog + endog + instruments
df['const'] = 1
iv_model = IV2SLS(
    dependent=df['earnings'],
    exog=df[['const']],
    endog=df[['college']],
    instruments=df[['log_distance']]
)
iv_result = iv_model.fit(cov_type='robust')
print(iv_result.summary)
```

输出摘要：

```
                         IV-2SLS Estimation Summary
────────────────────────────────────────────────────────────
Dep. Variable:                earnings   R-squared:      0.8253
Estimator:                      IV-2SLS   Adj. R-squared: 0.8251
No. Observations:                  1000   F-statistic:    166.22
────────────────────────────────────────────────────────────
             Parameter Estimates
─────────────────────────────────
            Params   Std.Err.   t-stat   p-value
─────────────────────────────────
const       2.3765    0.4696    5.060    0.0000
college     0.5045    0.0391   12.891    0.0000
─────────────────────────────────

Endogenous: college
Instruments: log_distance
```

**college 的系数 = 0.5045**——跟真实值 0.5 几乎重合。而且标准误（0.0391）比手动两阶段大了一点——这才是"诚实"的标准误。

## 🔗 延伸：因果推断在 AI 可解释性中的应用

你可能觉得"工具变量"是一个很经济学的东西——跟 AI 有什么关系？

关系很大。

### 问题：大模型的"回答"到底有没有因果效应？

想象一下你在用 ChatGPT 或 Claude：

```
用户提问 → AI 生成回答 → 用户觉得"这个回答有用"
```

问题来了：**用户觉得回答有用，到底是因为回答本身的质量，还是因为提问本身的质量？**

聪明的提问者往往能获得更高质量的回答（就像能力强的人更容易上大学一样）。所以如果用 OLS 直接估计"回答质量 → 用户满意度"：
```
满意度 = β₀ + β₁ × 回答质量 + ε
```
β̂₁ 会**有偏**——因为它把"好问题本身"的效应也算在了"好回答"的头上。

### 工具变量能做点什么？

在因果推断的 AI 可解释性研究中，研究者已经找到了多种"工具变量"的思路：

| 场景 | 内生变量 | 工具变量 | 直觉 |
|---|---|---|---|
| LLM 评估 | 回答质量 | 模型的温度参数 | 温度影响回答的多样性，但不直接影响用户的知识水平 |
| 推荐系统 | 用户点击 | 推荐位随机展示 | 位置影响点击概率，但不影响用户对该内容的真实兴趣 |
| 模型归因 | 特征重要性 | 输入特征的随机扰动 | 扰动改变特征值，但不改变数据本身的结构 |
| A/B 实验 | 是否为新版本 | 随机分组 | 分组是随机的，跟用户特征无关 |

以 LLM 为例：**你给同一个问题设置不同的 temperature，得到不同的回答，用户对这些回答的满意度差异——就是"回答质量"本身的因果效应**，因为提问内容在 temperature 变化时是固定的。

### 因果推断 + AI 的前沿方向

一种更前沿的思路叫**因果表示学习（Causal Representation Learning）**：

- 传统 AI 只看**相关性**——"这张图片里经常出现狗和草地，所以它们相关"
- 因果 AI 要学**因果结构**——"狗的出现是由动物存在导致的，草地背景跟狗无关"

工具变量在这里的作用是：**利用外部的不受模型控制的"随机变化"（如数据增强方式、噪声注入），来解耦特征之间的因果结构**。

比如你给一张狗的图片加高斯噪声：
- 狗的颜色变了 → 但"狗"这个概念没变
- 背景模糊了 → 但"狗"这个概念没变

这种**外生的扰动**，就像家到大学的距离一样——它改变了你观察到的某些特征，但是不影响潜在的因果结构。通过追踪这些变化，AI 可以逐步学会区分"因果"和"相关"。

## ✅ 小结

这一篇我们学到：

- **内生性问题**：当解释变量跟误差项相关时，OLS 是有偏的——"上大学回报率"被高估就是因为能力混杂
- **工具变量（IV）**：找一个变量 Z，它只通过内生变量 X 影响结果 Y——不直接、不混杂
- **两个核心条件**：**相关性**（Z 跟 X 相关）+ **排他性**（Z 只通过 X 影响 Y）
- **2SLS 两步法**：第一阶段用 Z 预测 X 得到 X̂；第二阶段用 X̂ 预测 Y——标准误需要用 linearmodels 一次性跑完
- **弱工具变量**：第一阶段 F < 10 时 IV 不可靠
- **AI 延伸**：因果推断在 LLM 评估、推荐系统、因果表示学习中越来越重要——工具变量是连接"相关性"和"因果性"的桥梁

# 断点回归——多一点就进了名校？

> 你的高考分数是 599，人家是 600。一分之差，你上了二本，他进了 985。这一分到底改变了你的人生多少？断点回归就是想回答这个问题。

---

## 🎯 一道分数线，两个世界

假设你是一名高三毕业生。

高考成绩出来了：**599 分。** 你的目标是某所 985 高校，它的投档线是 **600 分。**

差 1 分。你落榜了。

平行时空里的另一个你，考了 **601 分**——稳稳地录取了。

多年以后，两个人的收入、人脉、视野会不会天差地别？**或者说——这 1 分的"运气"到底值多少钱？**

这个问题，经济学家每天都在问。但他们没法做"随机实验"（总不能把考生随机分配到名校和普通学校吧？）。

所以，大自然给了他们一个**天然的实验**——高考分数线。

重点来了：**600 分附近的"好运气"几乎是随机的。** 一个人考 599 还是 601，很大程度上取决于考场状态、选择题蒙对的概率、甚至是当天吃没吃早饭。因此，分数线附近构成了一个"近似随机实验"——考 601 的人去了名校，考 599 的人没去，除此之外，他们在各项特征上没有系统性差异。

这就是 **Regression Discontinuity Design（RDD，断点回归）** 的精髓：**找一个人为的断点，在断点附近做因果推断。**

---

## 🔪 断点回归的直觉：切断连续性

### 从哪里"断"？

先想一个概念：**运行变量（Running Variable）**。

在高考的例子中，运行变量就是**高考成绩**。它在分数线处有一个"断点"（cutoff）：

```
                    ┌── 录取（处理组）
                    │
    未录取（控制组） │
    ┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┪┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄  → 高考成绩
                    600分
```

RDD 的核心思想是：**如果高考成绩除了决定录取外，不会在 600 分这个点发生跳跃，那么未来收入的"跳跃"就可以归因于"被名校录取"这个原因。**

换句话说，我们画的不是整条曲线，而是**断点附近两边的局部拟合线**。这两条线在断点处的高度差，就是因果效应。

![断点回归示意图](images/021-rdd-scatter.png)
*图 1：断点回归的核心图像。横轴是高考成绩（运行变量），纵轴是未来收入。虚拟数据中，600 分处有一个明显的"跳跃"——超过分数线的学生，未来的平均收入明显更高。两条拟合线告诉我们：在断点附近，因果效应就是这两条线的落差。*

### 两种类型的 RDD

1. **精确断点回归（Sharp RDD）**：分数超过某个阈值 100% 被录取。大多数教育研究属于这一类。
2. **模糊断点回归（Fuzzy RDD）**：超过阈值只是**增加**了被处理的概率，而非 100%。比如奖学金的申请——过了线也不一定拿到，但概率大很多。

本篇以 Sharp RDD 为主线。

---

## 📐 数学：RDD 本质上是一个"局部实验"

### 形式化

令：
- $X_i$ = 第 $i$ 个学生的高考成绩（运行变量）
- $c$ = 分数线（断点）
- $D_i = \mathbb{1}(X_i \ge c)$ = 是否录取（处理变量）
- $Y_i$ = 未来收入（结果变量）

我们关心的因果效应是：

$$\tau = \lim_{x \to c^+} E[Y_i | X_i = x] - \lim_{x \to c^-} E[Y_i | X_i = x]$$

翻译成人话：**断点右侧的收入极限，减去断点左侧的收入极限。** 这个差值就是因果效应。

### 怎么估计 $\tau$？

最直接的方法：在断点两边分别做回归。

**线性模型（一阶）**：

$$Y_i = \alpha + \tau D_i + \beta (X_i - c) + \gamma D_i \cdot (X_i - c) + \varepsilon_i$$

- $\tau$：断点处的跳跃（因果效应）
- $\beta$：分数线左边，成绩每高 1 分，收入的增长
- $\gamma$：分数线右边，斜率的变化（与左边不同）

**二次模型（二阶）**：加上 $(X_i - c)^2$ 和 $D_i \cdot (X_i - c)^2$，允许曲线弯曲。

关键点：**模型阶数越低，对带宽（bandwidth）越敏感；带宽越小，越逼近"局部实验"。**

### 带宽问题

我们把高考成绩±20 分以内叫"小带宽"，±50 分叫"大带宽"。

- 小带宽：更接近随机实验，但样本少、方差大
- 大带宽：样本多、方差小，但可能引入系统性偏差（高分考生本身能力更强）

因此，**带宽选择是 RDD 的核心权衡**。标准做法是：

1. 用 **MSE 最优带宽**（像 Imbens-Kalyanaraman 方法）
2. 做 **带宽敏感性分析**（用不同的带宽重复估计）
3. 报告结果是否稳健

![带宽敏感性分析](images/021-rdd-bandwidth.png)
*图 2：不同带宽下的 RDD 估计及其置信区间。横轴是带宽宽度，纵轴是估计的因果效应。一般来说，带宽越大，估计越精确（置信区间窄），但偏差越大。找到一个"甜区"——使得估计结果稳定的带宽范围。*

## 🐍 Python 实战：模拟高考断点数据

### 第一步：生成数据

我们模拟 10000 个考生的数据。运行变量是高考成绩，在 600 分处有一道"分数线"。

超过 600 分的学生被名校录取，他们的未来收入有一个"跳跃"（因果效应设为 5，单位为千元/月）。

```python
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
import statsmodels.api as sm
from numpy.polynomial import polynomial as P

np.random.seed(42)

n = 10000
# 高考成绩：300 ~ 750 (模拟全国卷大致范围)
score = np.random.uniform(300, 750, n)
cutoff = 600

# 录取状态
admitted = (score >= cutoff).astype(float)

# 基础收入：与成绩线性相关（每高 10 分收入+0.3）
income_base = 5 + 0.03 * (score - cutoff)

# 因果效应：超过分数线的人额外 +5（千元/月）
causal_effect = 5.0

# 误差项
epsilon = np.random.normal(0, 2.5, n)

# 最终收入
income = income_base + causal_effect * admitted + epsilon

df = pd.DataFrame({
    'score': score,
    'admitted': admitted,
    'income': income
})
```

### 第二步：画出 RDD 核心图——断点处的跳跃

```python
# 在断点两侧各取带宽 h 内的数据
h = 50  # 带宽

df_local = df[(df['score'] >= cutoff - h) & (df['score'] <= cutoff + h)]
df_left = df_local[df_local['score'] < cutoff]
df_right = df_local[df_local['score'] >= cutoff]

fig, ax = plt.subplots(figsize=(12, 7))

# 散点
ax.scatter(df_left['score'], df_left['income'], alpha=0.3,
           color='#4C9F70', s=8, label='未录取')
ax.scatter(df_right['score'], df_right['income'], alpha=0.3,
           color='#C73E1D', s=8, label='录取')

# 线性拟合（一阶）
for subset, color, label in [(df_left, '#2E86AB', '左拟合'),
                              (df_right, '#A23B72', '右拟合')]:
    X_local = sm.add_constant(subset['score'] - cutoff)
    m_local = sm.OLS(subset['income'], X_local).fit()
    xp = np.linspace((subset['score'] - cutoff).min(),
                     (subset['score'] - cutoff).max(), 100)
    yp = m_local.params[0] + m_local.params[1] * xp
    ax.plot(xp + cutoff, yp, color=color, linewidth=3, label=label)

# 标注跳跃
ax.annotate('因果效应\n≈ 5 千元/月', xy=(cutoff + 2, 10.5),
            xytext=(cutoff + 40, 13),
            fontsize=12, color='#C73E1D', fontweight='bold',
            arrowprops=dict(arrowstyle='->', color='#C73E1D', lw=2))

ax.axvline(x=cutoff, color='gray', linestyle='--', linewidth=1.5, alpha=0.6)
ax.set_xlabel('高考成绩', fontsize=13)
ax.set_ylabel('未来月收入（千元）', fontsize=13)
ax.set_title('断点回归：分数线附近的因果效应', fontsize=15, fontweight='bold')
ax.legend(fontsize=10)
ax.grid(alpha=0.2)
```

### 第三步：用不同带宽做敏感性分析

```python
bandwidths = np.arange(5, 101, 5)
estimates = []
cis = []

for h in bandwidths:
    sub = df[(df['score'] >= cutoff - h) & (df['score'] <= cutoff + h)]
    sub_left = sub[sub['score'] < cutoff]
    sub_right = sub[sub['score'] >= cutoff]

    # 左右分别做线性回归，在断点处取预测值
    # 左
    X_l = sm.add_constant(sub_left['score'] - cutoff)
    m_l = sm.OLS(sub_left['income'], X_l).fit()
    pred_l = m_l.params[0]  # 在 cutoff 处的预测值（X-c=0）

    # 右
    X_r = sm.add_constant(sub_right['score'] - cutoff)
    m_r = sm.OLS(sub_right['income'], X_r).fit()
    pred_r = m_r.params[0]

    tau = pred_r - pred_l
    estimates.append(tau)

    # 近似标准误（delta method 简化版）
    se = np.sqrt(m_l.bse[0]**2 + m_r.bse[0]**2)
    cis.append((tau - 1.96 * se, tau + 1.96 * se))

estimates = np.array(estimates)
cis = np.array(cis)
```

![带宽敏感性分析](images/021-rdd-bandwidth.png)
*图 2：不同带宽下的估计结果。横轴是带宽，纵轴是因果效应估计值，灰色区域是 95% 置信区间。真实的因果效应是 5（红线）。可以看到：带宽太小（<15）时方差很大，带宽太大（>80）时可能因样本偏差而偏离真实值。稳健的 RDD 应选择一个"平台期"——估计值稳定的带宽范围。*

### 第四步：密度检验——断点处有没有"操纵"？

RDD 有一个致命假设：**考生不能精确操纵自己的分数。**

如果考生能精确控制分数（比如通过作弊加分刚好到 600 分），那"600 分附近近似随机"的假设就崩塌了。

怎么检验？**看密度函数在断点处是否平滑。**

如果有人在断点右侧"攒"了很多样本（大家都把自己搞到 601 分），那密度在断点处就不是连续的——右边会多出一截。

这就是 **McCrary 密度检验** 的基本思想。

```python
# 简单实现：在断点两侧画直方图对比
fig, ax = plt.subplots(figsize=(10, 6))

# 断点两侧直方图
bins = np.linspace(cutoff - 50, cutoff + 50, 40)
ax.hist(df_left['score'], bins=bins[bins < cutoff],
        alpha=0.7, color='#4C9F70', label='左侧（未录取）',
        density=True)
ax.hist(df_right['score'], bins=bins[bins >= cutoff],
        alpha=0.7, color='#C73E1D', label='右侧（录取）',
        density=True)

ax.axvline(x=cutoff, color='gray', linestyle='--', linewidth=1.5)
ax.set_xlabel('高考成绩', fontsize=13)
ax.set_ylabel('密度', fontsize=13)
ax.set_title('McCrary 密度检验：断点处密度是否连续？', fontsize=14, fontweight='bold')
ax.legend(fontsize=10)
ax.grid(alpha=0.2)
```

![密度检验](images/021-rdd-density.png)
*图 3：断点两侧的密度分布。如果没人操纵分数，密度在断点处应该是平滑的（没有"跳跃"）。这张模拟图中，两侧密度在 600 分附近基本一致——说明我们不能拒绝"密度连续"的原假设，RDD 的假设未被违反。*

如果密度检验显示断点处密度不连续（比如左侧密度断崖式下降，右侧突然暴增），那 RDD 的有效性就值得怀疑——**考生可能操纵了自己的分数**。

## 📊 RDD 的完整工作流

```
1️⃣ 确认断点存在（Sharp or Fuzzy）
        │
2️⃣ 画断点图（散点 + 拟合线）
        │
3️⃣ 选择模型阶数（线性/二次）
        │
4️⃣ 选择带宽（MSE 最优 / 交叉验证）
        │
5️⃣ 做敏感性分析（不同带宽、不同阶数）
        │
6️⃣ 做密度检验（McCrary 检验）
        │
7️⃣ 报告结果：τ 的估计值 + 稳健性讨论
```

## 🔗 延伸：量化交易中的"断点"

断点回归不仅仅属于社会科学和医学。**它在量化交易中也有大量应用。**

### 调仓日效应

指数基金的调仓日（如 MSCI 每季度调整成分股）是一个天然的"断点"：

- **断点前**：被调入的股票可能被提前买入
- **断点后**：卖出压力释放

用 RDD 的思路来看调仓效应：
- 运行变量 = 股票在排名中的位置（比如在调入/调出边界附近）
- 断点 = 被划入/划出指数的那条线
- 结果 = 调仓日附近的收益率

研究发现，**调仓日在断点处确实存在显著的"跳跃"**——被调入的股票平均上涨 1-2%，被调出的下跌 1-2%。这个效应短期内是显著的，但长期会回归。

### 打新效应

新股上市第一天，中签的人卖出，没中签的人买入。中签的"断点"分配也类似于 RDD——**打新资金量稍微多一点就能中签**，这构成了一个天然的断点。

### 因子投资中的断点思维

很多因子策略本质上利用了断点思维：
- 分位数分组的策略（前 30% 做多，后 30% 做空）
- 在分界线附近的股票具有相似的暴露度
- **分组后的差异化收益，部分源于断点处的随机性分配**

## ✅ 小结

这一篇我们学到：

- **RDD 是什么**：利用人为设定的断点，在断点附近做"近似随机实验"
- **核心直觉**：断点两侧的跳跃 = 因果效应
- **两种类型**：精确断点（Sharp）和模糊断点（Fuzzy）
- **关键选择**：带宽大小——方差 vs 偏差的权衡
- **检验方法**：密度检验确保没有人操纵运行变量
- **量化延伸**：调仓日效应、打新效应都是 RDD 的应用场景

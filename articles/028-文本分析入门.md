# 把文字变成数字——文本分析的计量入门

> 好评如潮的商品一定卖得好吗？"还不错"和"太棒了"在数学模型里差多少分？今天我们把中文评论变成数字，然后跑一个回归。

---

## 📱 一个小问题：怎么知道用户喜不喜欢你的产品？

假设你在淘宝上卖零食。每天有几十条评论涌进来：

- "太好吃了！回购三次了，推荐给大家～"
- "一般吧，没有想象中好吃，价格也有点贵。"
- "物流太慢了，等了五天，东西也碎了，差评！"

你一条一条地看，大概能判断——嗯，今天风评不错，或者今天被骂了。

但问题来了：

**👉 如果一个月有 3,000 条评论，你还看得过来吗？**

更重要的是：**用户的评价和销量之间到底有没有关系？有多大关系？** 换句话说——

> 情感每提高 1 分，销量能涨多少？

要回答这个问题，你得先**把文字变成数字**。这就是文本分析（Text Analysis）做的事。

---

## 🔤 第一步：从文字到数字——直觉

把文字变成数字，听起来很玄乎，但逻辑其实很简单。

### 最朴素的方法：词袋模型（Bag of Words）

假设一条评论是 "好吃，物流快，推荐"。

我们把这句话拆成词语：`["好吃", "物流", "快", "推荐"]`。

然后数每个词出现的次数：

| 好吃 | 物流 | 快 | 推荐 | 差 | 贵 | 失望 |
|------|------|----|------|-----|-----|------|
| 1    | 1    | 1  | 1    | 0   | 0   | 0    |

这条评论就变成了一个**数字向量**。就这么简单。

### 情感分析——更进一步

词袋模型把所有词一视同仁。但"好吃"和"难吃"显然含义完全相反。

**情感分析**就是给词贴标签：

- 正面词（+1）：好吃、满意、推荐、划算……
- 负面词（-1）：差、贵、失望、难吃……

然后一条评论的情感得分 = （正面词数 - 负面词数）/ 总词数。

比如：
- "好吃，物流快，推荐" → 全是正面词，得分很高 ✅
- "价格贵，东西也一般，失望" → 负面词占多数，得分很低 ❌
- "还行吧" → 没有情感词，得分 ≈ 0 😐

**情感得分 → 数字 → 放进回归模型。** 这就是文本分析的计量玩法。

## 🧪 Python 实战：模拟商品评论的情感分析

真实数据需要爬虫，今天我们**模拟**一套产品评论数据，完整走一遍"文字→数字→回归"的流程。

### 第一步：构建情感词典

首先我们要手工定义一组"好词"和"坏词"：

```python
# 正面词示例
positive_words = [
    '好吃', '满意', '推荐', '赞', '不错', '实惠', '新鲜',
    '喜欢', '值得', '划算', '方便', '良心', '惊喜', '超值'
]

# 负面词示例
negative_words = [
    '差', '一般', '失望', '慢', '贵', '难吃', '差评', '不推荐',
    '敷衍', '变质', '后悔', '不值', '垃圾', '坑', '差劲'
]
```

这就是一个**基于词典的情感分析**方法。现实中有更成熟的词典（知网 Hownet、BosonNLP 情感词典），原理都是类似的。

每个正面词 +1 分，每个负面词 -1 分。

### 第二步：模拟评论 + jieba 分词

我们生成 200 条模拟中文评论，每条评论由模板 + 随机情感词构成。

然后使用 **jieba**（结巴分词）把每句话切成词语列表：

```python
import jieba

text = "这家店的东西真的好吃，物流快，推荐！"
words = jieba.lcut(text)
# 输出: ['这家', '店', '的', '东西', '真的', '好吃', '，', '物流', '快', '，', '推荐', '！']
```

再跟情感词典比对，计算每条评论的情感得分。

### 第三步：模拟销量 + 回归

现在我们有了"情感得分"。再模拟销量数据——销量受情感得分、价格、促销活动影响：

```
销量 = 500 + 80 × 情感得分 - 1.5 × 价格 + 120 × 促销活动 + 噪声
```

然后跑一个多元回归：

```python
import statsmodels.api as sm

X = sm.add_constant(df[['sentiment_score', 'price', 'promotion']])
m = sm.OLS(df['sales'], X).fit()
print(m.summary())
```

输出结果大概是这样（每次运行略有不同）：

```
==========================================================================
                 coef    std err          t      P>|t|      [0.025      0.975]
--------------------------------------------------------------------------
const        510.2343     27.454     18.581      0.000     456.141     564.328
sentiment     82.1564     10.237      8.025      0.000      62.013     102.300
price         -1.4872      0.212     -7.015      0.000      -1.905      -1.069
promotion    118.3412     12.156      9.735      0.000      94.412     142.270
==========================================================================
R-squared:                        0.684
==========================================================================
```

关键解读：
- **情感得分每提高 1 分，销量平均增加约 82 件**（p < 0.001，显著）
- **价格每上涨 1 元，销量下降约 1.5 件**（正常的经济学直觉）
- **促销活动能让销量提高约 118 件**
- **R² = 0.684**——这些变量能解释销量变异的 68.4%

情感分析的计量意义就在这里：**你把非结构化的文本量化为数字，然后可以像普通变量一样放进你的回归模型，回答"这个文字特征到底有多大的经济效应"。**

## 🔗 延伸：NLP 在量化基本面分析中的应用

你可能觉得"文本分析"只是电商平台的事。其实它在**量化投资**中也是一个越来越热门的领域。

### 新闻情绪因子

量化研究员用 NLP 分析财经新闻、公司公告、社交媒体帖子中的情绪，构建**新闻情绪因子**：

- 正面新闻多的公司 → 情绪因子高 → 预期超额收益
- 负面新闻集中的行业 → 情绪因子低 → 减仓信号

**经典文献**：Tetlock (2007, JFE) 发现《华尔街日报》的"市场对话"专栏中负面词汇比例能预测道琼斯指数的下跌。

### 财报电话会议文本

公司每个季度开财报电话会议，高管发言的文字记录里藏着大量信息：

- **管理层语调**：乐观还是悲观？用情感分析量化
- **可读性**：句子太复杂可能是在"隐藏坏消息"（Li, 2008）
- **回答回避度**：分析师提问时，高管答非所问的程度

这些文本特征都可以变成**横截面选股因子**——放进 Fama-MacBeth 回归里，和市盈率、市净率放在一起。

### 中文金融情感词典

做中文金融文本分析，不要直接用通用情感词典。"亏损"在金融文本里不是负面情绪而是事实陈述。有专门的中文金融情感词典（比如姜富伟等，2021 在《管理科学学报》发布的金融情感词典），更适合分析 A 股公告和新闻。

## ⚠️ 词典法的三个局限

我们这篇用的是**基于词典的情感分析**（Lexicon-based Approach）。它简单、可解释、适合教学。但你得知道它的不足：

**1. 否定词和程度副词**

- "好吃" = +1
- "不好吃" = ？词典法可能还是 +1（因为"不"不在情感词典里，"好吃"在）
- 修复方案：加否定词规则（"不""没"等反转情感极性）

**2. 上下文依赖**

- "这个产品真的**垃圾**" → 负面 ✅
- "服务器**垃圾**回收概念股大涨" → 中性 ❌（"垃圾"在这里是行业属性）

**3. 网络新词和领域术语**

"yyds"、"绝绝子"、"破防了"——这些词在最新的评论里大量出现，但传统词典里没有。需要不断更新词典。

**进阶方法**：机器学习的文本分类（用 sklearn 的朴素贝叶斯或 SVM 训练一个情感分类器），或者用预训练模型（BERT）。但这篇不展开——我们的目标是**把文字变成数字做回归**，词典法已经把这个逻辑讲透了。

## ✅ 小结

这一篇我们学到：

- **文本分析的计量逻辑**：把非结构化的文字变成数字 → 放进回归模型 → 回答经济问题
- **情感分析三步骤**：构建词典 → 分词匹配 → 计算得分
- **工具链**：jieba（分词） + pandas（数据处理） + statsmodels（回归）
- **延伸**：NLP 在量化投资中可以构建新闻情绪因子、分析财报语调
- **局限**：词典法简单但粗糙，不处理否定、歧义、新词

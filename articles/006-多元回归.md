# 不止一个变量：多元回归——控制其他变量，看净效应

> 如果你只用一个变量做回归，你的结论可能是错的。因为——还有别的变量在同时起作用。

---

## 🧩 一个现实问题

上一篇我们用房价数据做了一次完整分析，这里我们专门聚焦一个核心问题。

假设你想知道**面积对房价的真实影响**。

你跑了一个简单回归：房价 = β₀ + β₁ × 面积。结果 β₁=2.73——**每大 1 m²，贵 2.73 万**。

但一个朋友说："大房子通常位置也更好，位置好本身就能推高房价。你的 2.73 万/m²，可能有一部分是位置的功劳。"

他说得对。这就是我们第 3 篇讲的**遗漏变量偏误**。

那怎么办？把位置放进去！

---

## 📐 多元回归：从"线"到"平面"

简单回归画出一条线。多元回归画出一个**平面**（或者超平面，当你有很多变量时）。

当我们同时放入面积和位置时：

```
房价 = β₀ + β₁×面积 + β₂×位置 + ε
```

β₁ 的含义变了：**位置相同的情况下**，面积每大 1 m²，房价会涨多少。

这就是"控制其他变量"（holding other variables constant）——多元回归最强大的能力。

![3D 回归平面](images/006-3d-plane.png)
*图 1：面积和位置两个变量共同解释房价。数据点是散在三维空间中的，橙色的半透明平面是回归拟合的结果。平面的"斜率"在两个方向上各自代表了面积的效应和位置的效应。*

## 🔍 把"控制"可视化出来

这里有一个非常直观的方法来理解"控制"——**偏回归图（Added-Variable Plot）**。

做法很简单：

1. 先用所有其他变量（位置、房龄、卧室、楼层）预测**房价**，得到残差——这是房价中不能被这些变量解释的部分
2. 用同样的变量预测**面积**，得到残差——这是面积中不能被这些变量解释的部分
3. 把这两个残差画成散点图

**你看到的就是：剔除其他所有因素的干扰后，面积对房价的"纯效应"。**

```python
import statsmodels.api as sm
import pandas as pd

# 第一步：价格中不能被其他变量解释的部分
X_others = sm.add_constant(df[['location', 'age', 'bedrooms', 'floor']])
resid_price = sm.OLS(df['price'], X_others).fit().resid

# 第二步：面积中不能被其他变量解释的部分
resid_area = sm.OLS(df['area'], X_others).fit().resid

# 第三步：看这两个残差之间的关系
m_partial = sm.OLS(resid_price, sm.add_constant(resid_area)).fit()
print(m_partial.params.iloc[1])
```

结果：**2.76**——和多元回归中面积的系数完全一致。

![偏回归图](images/006-partial-regression.png)
*图 2：控制了位置、房龄、卧室、楼层之后，面积的"净变异"和房价的"净变异"之间的关系。这条线的斜率=多元回归中面积的系数。*

## ⚖️ 简单回归 vs 多元回归：差距有多大？

直接看对比：

![系数对比](images/006-simple-vs-multi.png)
*图 3：左图是面积的系数对比。简单回归的 2.73 和多元回归的 2.76 差别不大——面积的真实效应很稳健。右图是位置的系数对比：在简单回归中位置的效应被低估了 1.47（4.68 → 6.15），因为面积和位置存在正相关——大房子往往位置也好。*

| 变量 | 简单回归 | 多元回归（控制其他） | 变化 |
|------|:-------:|:-----------------:|:----:|
| 面积（万/m²） | 2.73 | 2.76 | +0.03 |
| 位置（万/分） | 4.68 | 6.15 | **+1.47** |
| R² | 0.845 | 0.881 | +0.036 |

**为什么位置的系数变了这么多？** 因为：

简单回归（只放位置）：β₂ = 4.68 → 说"每提高 1 分位置评分，房价涨 4.68 万"
多元回归（控制面积）：β₂ = 6.15 → 说"在面积相同的情况下，每提高 1 分位置评分，房价涨 6.15 万"

真实的关系是：好位置+大面积往往同时出现。如果不控制面积，位置的效应看起来比真实的低——因为位置好推高房价的同时，位置好的房子面积也大，面积的效应混入位置的系数中被"拉低"了。

## 🤔 那到底该放多少变量？

一个常见的直觉错误是：**变量放得越多越好。** 不对。

- 放少了 → 遗漏变量偏误（第 3 篇）
- 放多了 → 可能引入**坏的控制变量**

哪些变量不该放？

1. **中介变量**（Mediator）：如果你想知道"上大学对收入的影响"，"毕业后的第一份工作"就是中介变量——上大学 → 找到好工作 → 高收入。控制"第一份工作"会挡住"上大学"的真实效应。
2. **碰撞因子**（Collider）：如果你控制了"面试表现"，而面试表现受到"能力"和"面试官心情"的共同影响——那控制它会创造出虚假的相关。
3. **无关变量**：加再多不影响系数，但浪费自由度。

**一个简单的判断原则：** 放那些同时影响 X 和 Y 的变量（混杂变量）。不放那些被 X 影响的变量（中介变量）。

## 🔗 延伸：从多元回归到 Fama-French 三因子模型

多元回归在量化金融中有一个非常著名的应用——**Fama-French 三因子模型**。

```python
股票超额收益 = α + β₁×市场因子 + β₂×规模因子 + β₃×价值因子 + ε
```

这里的 β₁、β₂、β₃ 就是多元回归的系数，含义分别是：
- **β₁（市场 Beta）**：大盘涨 1%，这支股票平均涨 β₁%。控制了规模和估值因素后。
- **β₂（规模因子）**：大盘每涨 1%、估值因素不变时，小盘股相对大盘股的额外收益。
- **β₃（价值因子）**：控制了市场和规模后，价值股相对成长股的额外收益。

这个模型的精髓，和本文完全一样——**"控制其他因素后，看单独一个因素的净效应。"**

## ✅ 小结

这一篇我们学到：

- **多元回归** = 同时放入多个解释变量，让每个变量的系数都是"控制其他变量不变"后的净效应
- **3D 到超平面** — 两个变量是平面，三个以上是超平面，思想一样
- **偏回归图** — 用残差把"控制"可视化出来
- **不是放越多越好** — 要选对变量，避免中介变量和碰撞因子
- **Fama-French 三因子** — 就是多元回归在量化金融中的应用

# 数据有了时间标签——自相关与ACF

> 今天的温度和昨天的有没有关系？明天的价格会不会受今天的影响？当数据带上时间标签，一个全新的世界打开了。

---

## 🌡️ 北京今天的温度和昨天有关吗？

先问你一个直白的问题：**明天北京的最高气温，和今天的最高气温有关系吗？**

你的直觉肯定告诉你：有关系。大概率明天的气温不会和今天差太多——尤其是春秋过渡季节，气温是"慢慢变"的，不是"蹦迪式"跳变的。

但如果我换一个问题：**明天北京的空气质量指数 AQI，和今天的 AQI 有关系吗？**

你的直觉同样会告诉你：有关系——雾霾不是一夜散尽的。

再换一个：**明天贵州茅台的收盘价，和今天的收盘价有关系吗？**

嗯……这个答案没那么简单。价格当然会受今天的影响，但**如果所有人都知道明天会涨，那今天就会涨到位**——这是金融学里最微妙的张力。

这一篇，我们就来正式认识**时间序列数据**——也就是那些带上了时间标签的数据。以及它的核心特征：**自相关**。

---

## 🔗 什么是自相关？

在标准回归中（第 1-13 篇），我们处理的基本是**横截面数据**：

- 100 个消费者的收入 × 消费（第 10 篇）
- 500 套房子的面积 × 价格（第 5 篇）

每一行是一个独立的个体。A 的消费和 B 的消费之间，没有系统性的关联。

但**时间序列数据**不一样。每一行是一个**时间点**，前后之间天然有联系。

```
t=1： 温度 15°C
t=2： 温度 16°C  ← 跟 t=1 有关系
t=3： 温度 17°C  ← 跟 t=2 有关系
...
```

我们把变量与其自身**滞后值**之间的相关性，叫做**自相关**（Autocorrelation）。

**自相关（Autocorrelation）** = 一个序列和它的"昨天"（滞后一期）的相关性。

```
ρ₁ = Corr(y_t, y_{t-1})   ← 一阶自相关（lag 1）
ρ₂ = Corr(y_t, y_{t-2})   ← 二阶自相关（lag 2）
...
ρₖ = Corr(y_t, y_{t-k})   ← k 阶自相关
```

当 ρ₁ 显著不为零时，就说明"昨天的 y 能预测今天的 y"。

> 听起来很简单对吧？但自相关是时间序列分析的基石——它直接决定了：
> - 你的**标准误**该怎么算（还记得第 10 篇的 HAC 吗？）
> - 能不能用过去预测未来
> - 一个"看起来显著"的回归结果，到底是真的还是**伪回归**

---

## 📐 自相关函数（ACF）

皮尔逊相关系数你肯定不陌生了（第 3 篇）：

```
Corr(X, Y) = Cov(X, Y) / (σ_X · σ_Y)
```

自相关函数（Autocorrelation Function，ACF）就是把同一个序列在不同滞后阶数 k 下的自相关系数排成一排：

```
ACF(k) = Corr(y_t, y_{t-k})
```

对于 k = 0，ACF(0) = 1（自己和自己当然完全相关）。所以我们一般从 k = 1 开始看。

**ACF 图**就是横轴是滞后阶数 k，纵轴是 ACF(k) 的柱状图。每根柱子告诉你：**y_t 和 y_{t-k} 的相关性有多大**。

上面还会画两条蓝色的置信区间带（通常是 ±2/√n）。柱子超出蓝带 → 该阶自相关是统计显著的。

## 🐍 Python 实战：模拟气温数据并画 ACF

### 第一步：生成带季节性的气温数据

一年 365 天，北京的气温大致是正弦波——夏天热冬天冷，加上随机扰动。

```python
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
from statsmodels.tsa.stattools import acf
from statsmodels.graphics.tsaplots import plot_acf

np.random.seed(42)
n_days = 365 * 3  # 3 年数据
t = np.arange(n_days)

# 正弦波模拟气温：均值 15°C，振幅 15°C（夏冬温差约 30°C）
temperature = (
    15 + 15 * np.sin(2 * np.pi * t / 365 - np.pi/2)
    + np.random.normal(0, 3, n_days)
)
```

![模拟的三年日均气温](images/014-temperature-timeseries.png)
*图 1：模拟的北京三年日均气温。看到清晰的"年周期"了吗？夏天波峰、冬天波谷——今天的温度和昨天的有关，这是最直观的自相关。*

### 第二步：画出 ACF

```python
fig, ax = plt.subplots(figsize=(10, 5))
plot_acf(temperature, lags=40, zero=False, ax=ax)
ax.set_xlabel('滞后阶数（天）')
ax.set_ylabel('自相关系数')
ax.set_title('ACF：气温的自相关函数')
```

![ACF 图](images/014-acf-example.png)
*图 2：气温的 ACF——lag 1 的自相关系数接近 0.95，随滞后阶数增加缓慢下降。"缓慢下降"本身就是季节性的标志。*

这个 ACF 图有什么特点？

**它不会像白噪声那样"唰"地降到零——它是慢慢下降的。** Lag 1 高达 ~0.95，lag 2 也接近 0.9，到 lag 30 还有 0.4 左右。

为什么？因为气温变化是慢变量——今天热，明天大概率还是热的；夏天的最热的那天，和 30 天后的气温仍然高度相关。

**ACF 缓慢衰减 ≈ 序列具有强自相关 ≈ 可能有趋势或季节成分。**

## ⚪ 白噪声 vs 自相关序列

什么叫"没有自相关"？我们看看**白噪声**。

白噪声（White Noise）是时间序列中最干净的情况——每一期的值都和过去无关，纯粹是随机波动。

```python
white_noise = np.random.normal(0, 1, n_days)
```

我们来对比一下：

![白噪声 vs 自相关序列对比](images/014-white-noise.png)
*图 3：上面是有自相关的气温序列，下面是白噪声。你能一眼看出区别——一个"平滑"，一个"毛刺"。ACF 图更是天差地别：白噪声的 ACF 几乎全部落在置信区间内（没有显著的自相关）。*

白噪声的 ACF：
- Lag 1 的自相关系数 ~0.02（不显著）
- Lag 2 也 ~-0.01（不显著）
- 绝大多数柱子都在蓝色置信区间以内

**白噪声的 ACF ≈ 除了 lag 0 外全部为零。**

所以当你看到 ACF 图上大部分柱子都"冒出头"（超出蓝带），就说明这个序列不是白噪声——它有"记忆"。

## 💹 延伸：股票价格的"特殊气质"

说完气温和白噪声，我们来看看**股票价格**。

股票价格不是用正弦波模拟的——它是用**随机游走**（Random Walk）来模拟的：

```python
# 随机游走：每一步是前一步 + 随机冲击
returns = np.random.normal(0.001, 0.02, n_days)  # 日收益率
price = 100 * np.exp(np.cumsum(returns))  # 价格是收益率的累积
```

![模拟股票价格](images/014-stock-price.png)
*图 4：模拟的股票价格走势。它和气温有什么不同？看 ACF——气温的 ACF 缓慢下降，而价格的 ACF 也是缓慢下降。但两者的**本质**完全不同。*

这里有一个关键区别——**气温是平稳的**，**股价是非平稳的**。

- **气温**：围绕 15°C 的均值波动，方差有限。你长期预测气温，总是回归到均值附近。
- **股价**：没有固定的均值，方差随时间增长。你长期预测股价……没有人知道它会飘到哪里。

**但这还不是最有趣的。** 最有趣的是**收益率**——也就是价格的日变化率：

```python
returns = np.diff(np.log(price))  # 对数收益率
```

收益率的 ACF 往往是这样的：
- Lag 1 的自相关系数 ~0.01~0.05（非常接近零）
- 基本上和白噪声差别不大

**这意味着：价格有高度自相关（非平稳导致的伪相关），但收益率几乎没有自相关（至少在有效市场下）。**

这正是**有效市场假说**的计量基础：
- 价格不可预测（因为收益率是白噪声）
- 但价格的波动率可以预测（这就是第 18 篇 GARCH 的内容）

> **一个小思考**：如果收益率的 ACF 真的显著不为零（比如 lag 1 的 ρ₁ = 0.1），你就找到了一个可以赚钱的策略——今天涨了，明天还有小概率继续涨。但问题是，这个"0.1"是真的规律还是数据挖掘的假象？——这就轮到第 16 篇的"平稳性"登场了。

## ✅ 小结

这一篇我们学到：

- **时间序列数据** = 带时间标签的数据，前后相关是常态
- **自相关** = 变量与其滞后值的相关性，用 ACF 来衡量
- **ACF 图** = 横轴滞后阶数，纵轴自相关系数，蓝色区间 = 显著性边界
- **白噪声** = 没有自相关的"纯随机"序列，ACF 几乎全部在蓝带内
- **气温是平稳的**：围绕均值波动，ACF 缓慢衰减
- **股价是非平稳的**：没有固定均值，ACF 看似也缓慢衰减，但原因是不同的
- **收益率 ≈ 白噪声**：有效市场假说的计量对应

**下一篇预告**：趋势和季节——把时间序列拆解成"长期趋势 + 季节效应 + 随机波动"三部分，看得更清楚。

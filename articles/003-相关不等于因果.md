# 相关不等于因果，可是为什么呢？——三大搅局者

> 上一篇文章我们学会了如何画一条"最好"的线。但这条线只告诉我们"X 和 Y 有关系"——它没有告诉我们"X 是不是 Y 的原因"。

---

## 📱 一个你每天都在面对的困惑

你看着手机上的屏幕时间统计数据：这周平均每天刷了 5 小时手机。同时你注意到——最近睡得不太好。

于是你产生了一个念头：**是不是刷手机太多导致我睡不好？**

但你有没有想过另一种可能性：你最近有个项目 deadline，压力很大。压力让你难以入睡，也让你更想刷手机逃避。刷手机和睡不好**同时发生**，不是因为手机导致了失眠，而是**压力导致了这两件事**。

这就是计量经济学最核心的一句话：**相关不等于因果。**

这一篇我们就来讲清楚：当两个变量一起变化的时候，为什么不能说一个导致了另一个？

---

## 🎭 三大搅局者

有两种东西能让你看到一个"假的相关"：

### 1. 反向因果 —— 到底是公鸡还是太阳？

公鸡打鸣和太阳升起几乎同时发生。如果你只看到数据，会发现打鸣几乎完美预测日出。但你能说"公鸡打鸣导致太阳升起"吗？

在数据里，X 和 Y 谁导致了谁，方向不一定是你想的那样：

- **你以为是**：刷手机 → 睡不好
- **也可能是**：睡不好 → 刷手机（睡不着当然刷）
- **甚至可能是**：两者互相强化（双向因果）

### 2. 遗漏变量 —— 那个你看不到的第三者

这是计量里最重要、也最容易被忽视的问题。

如果有一个你**没测量的变量**同时影响了 X 和 Y，那么 X 和 Y 之间就会产生一个"假的相关"，这个假相关不是你想要的因果效应。

比如：
- 你没测量"压力"
- "压力"让人刷手机（为了逃避）
- "压力"也让人睡不好
- 于是数据上看到：刷手机越多=睡得越差
- 但这不是刷手机造成的！这是压力同时导致的

### 3. 纯噪声 —— 巧合而已

如果重复 1000 次随机实验，总会有那么几次两个完全无关的东西表现出了"统计显著"的相关。这就是多重比较谬误——你试得足够多，总有一个碰巧相关。

## 🔬 重点解剖：遗漏变量

遗漏变量是三种中最阴险的——因为它不声不响地污染了你的结论，而且几乎不可能从数据本身发现（除非你有额外的信息）。

来看一个具体的模拟：

假设有 100 个人，他们的**压力水平不同**。压力高的人刷手机多，睡眠也差。压力低的人刷手机少，睡眠也好。

我们作为研究者，只收集到了"刷手机时长"和"睡眠质量"的数据——我们不知道每个人的压力水平。

```python
import numpy as np
import matplotlib.pyplot as plt
import statsmodels.api as sm

# 真实的因果结构：压力(Z) → 刷手机(X), 压力(Z) → 睡眠质量(Y)
np.random.seed(42)
n = 100
stress = np.random.normal(0, 1, n)  # 这个变量我们没测量
phone_hours = 3 + 1.5 * stress + np.random.normal(0, 0.5, n)
sleep_quality = 8 - 1.2 * stress + np.random.normal(0, 0.5, n)

# 我们只看到 X 和 Y
X = sm.add_constant(phone_hours)
model_biased = sm.OLS(sleep_quality, X).fit()
print(model_biased.params['x1'])  # 输出：-0.67
```

看！回归告诉我们："每多刷 1 小时手机，睡眠质量下降 0.67 分。"

但这个结论是错的。真正的因果链条是：

```
压力 →（+）→ 刷手机
压力 →（-）→ 睡眠质量
```

刷手机和睡眠质量之间**本来没有直接的因果关系**。那个 -0.67 完全是"压力"这个遗漏变量制造出来的假象。

**如果我们能控制压力：**

```python
# 先去除压力对两者的影响
res_phone = sm.OLS(phone_hours, sm.add_constant(stress)).fit().resid
res_sleep = sm.OLS(sleep_quality, sm.add_constant(stress)).fit().resid

# 再跑回归
model_correct = sm.OLS(res_sleep, sm.add_constant(res_phone)).fit()
print(model_correct.params.iloc[1])  # 输出：-0.01 ≈ 0
```

控制压力后，刷手机对睡眠的影响从 -0.67 跌到了 ≈0。**那个"显著"的效应消失了。**

![遗漏变量的魔力](images/003-omitted-variable.png)
*图 2：红色实线是"不考虑压力"时的回归线（β = -0.67），看起来刷手机和睡眠差存在强负相关。一旦控制"压力"这个遗漏变量，效应几乎归零。*

## 📈 纯噪声：两个随机序列的"爱情故事"

接下来这个案例更震撼。

我们模拟两个完全独立的随机游走时间序列——一个叫"巧克力消费指数"，一个叫"诺贝尔奖指数"。它们之间没有任何关系，各自独立生成。

```python
np.random.seed(2023)
n = 100
chocolate = np.cumsum(np.random.normal(0, 1, n)) + 100
nobel = np.cumsum(np.random.normal(0, 1, n)) + 50
```

然后我们跑一个回归，看"巧克力消费"能否预测"诺贝尔奖"。

结果：

```
R² = 0.00, p = 0.71
```

等等——这一轮它不显著。那我们再跑一次？再换一个随机种子？

关键是：**如果只展示一次跑出来的"显著"结果，读者看到的是一段伪装成洞见的噪声。** 很多"回测策略""投资圣杯"就是这么来的——在 1000 个尝试中，有那么几个碰巧显著，然后拿出来当圣杯宣传。

![伪回归](images/003-spurious-regression.png)
*图 3：左上两条线走势相似——但它们是两个独立的随机游走，没有任何关系。如果只看"走势一致"就下结论说有关系，那就中招了。*

## 🔗 量化交易里的"相关不是因果"

这是最实用的一节。

量化交易中最大的陷阱之一就是**数据挖掘偏差**：你在几千个候选变量里来回测试，总能找到几个和收益率"显著相关"的因子。

比如说，你发现"过去 5 天中，有 3 天股价收涨时买入，持有 2 天卖出"的策略在回测中年化收益 30%。你兴奋地准备实盘。

等一下。你有没有想过：

- **反向因果**：不是这个信号导致涨，而是涨了才出现这个信号？
- **遗漏变量**：宏观经济环境同时在驱动股市和这个信号？
- **纯噪声**你在 500 个候选策略里试出来的这一个？如果是，它在样本外几乎肯定失效。

**计量经济学给你的是纪律**：它不是让你的策略更聪明，而是让你问出"你凭什么觉得这个关系是真的？"——然后给你一套方法来回答这个问题。

同样地，AI 领域的"可解释性"（XAI）本质上也在做同一件事：在复杂的神经网络中找出"哪些输入真的在驱动输出"，而不是看数据中的表面相关。

## ✅ 小结

这一篇我们学到：

- **反向因果** — X 导致 Y，还是 Y 导致 X？数据不会告诉你方向
- **遗漏变量** — 看不见的第三变量制造假相关，这是计量里最大的敌人
- **纯噪声** — 尝试得足够多，总会碰巧显著
- **DAG（因果图）** — 用箭头画出变量间的因果关系，让假设变得透明
- **控制变量** — 加入潜在的混杂变量，排除干扰

**为什么第一篇要用咖啡和代码行数的例子？** 因为：即使跑出了 β₁=75、p=0.000、R²=0.59，我们依然不能说"咖啡因提高生产力"——因为"状态好"这个遗漏变量没被控制。

# 计量+量化——用回归造一个选股因子

> 你在 5000 只股票里寻找赚钱的信号。财务指标成百上千——为什么有些"有效"，大部分"无效"？答案藏在一行回归里。

---

## 🎯 选股因子 = 用回归回答一个简单问题

想象你是基金经理，管理一个 50 亿的 A 股账户。

你的研究员抱着一堆数据冲进来："我发现一个规律——市净率低的股票，未来一个月收益更高！"

**你该怎么判断这个规律是真的，还是巧合？**

这就是本篇要解决的问题：**如何用计量经济学的方法，严格地从数据中"造"出一个选股因子，并检验它是否真的有效。**

整个流程就三步，和做医学实验几乎一样：

1. **提出假设**：某个特征（低市净率、高动量、低波动）能预测未来收益
2. **构建因子**：用回归量化这个预测能力——股票对这个特征的"暴露"能带来多少收益
3. **验证因子**：分层回测 + IC 分析 + Fama-MacBeth 检验——用三种方法交叉验证

我们一步一步来。

---

## 🧪 第一步：模拟数据——500 只股票，24 个月

在实际工作中，你会从 Wind/CSMAR 等数据源拉取数据。今天我们直接模拟，这样你可以看到数据生成的"真实因果"是什么，再回头看回归是否能把它找回来。

```python
import numpy as np
import pandas as pd

np.random.seed(42)
N, T = 500, 24  # 500 只股票，24 个月

# 真实的因子暴露（每个股票一个，不随时间变化）
beta_true = np.random.normal(0, 1, N)

# 因子收益率（时间序列，每月一个值）
f_t = np.random.normal(0.8, 0.5, T)  # 因子月均收益 0.8%

# 生成股票收益
returns = np.zeros((N, T))
for i in range(N):
    for t in range(T):
        returns[i, t] = (
            0.5                                    # 无风险基准
            + beta_true[i] * f_t[t]                # 因子贡献
            + np.random.normal(0, 2.0)              # 特质波动
        )
```

这里的关键设计：**我们人为制造了一个"真实因子"**。每个股票 i 有一个因子暴露 βᵢ（可以理解为"对因子的敏感度"），每月因子收益率 fₜ 不同。股票的收益 = 截距 + βᵢ × fₜ + 噪声。

计量学的任务是：**只知道收益数据，能不能把这个因子"挖"出来？**

---

## 📐 第二步：Fama-MacBeth 两步法（1973）

Fama 和 MacBeth 在 1973 年提出了一种方法，至今仍是金融学术界的金标准。它的核心思想非常优雅：

### 第一步：每个月跑一个横截面回归

```
Rᵢₜ = β₀ₜ + β₁ₜ × X₁ᵢₜ + β₂ₜ × X₂ᵢₜ + ... + εᵢₜ
```

横截面回归——在同一个时间点，用所有股票的特征来预测收益。

每个时间点 t，我们都得到一个系数 β̂ₜ。如果这个因子真的有效，β̂ₜ 应该**随时间稳定地不等于 0**。

### 第二步：对 β̂ₜ 做时间序列平均

```
β̂ = (1/T) × Σ β̂ₜ
t(β̂) = β̂ / (σ(β̂ₜ) / √T)
```

关键看 t 统计量——**它衡量的是因子收益率的"信号"是否大于"噪声"**。

> **Fama-MacBeth 的聪明之处**：传统的 OLS 对面板数据直接回归会低估标准误（因为同一期的股票残差相关）。Fama-MacBeth 先跑 T 个横截面回归，然后对系数做时间序列的 t 检验——这自动考虑了截面相关性。不需要复杂的 cluster 标准误，非常优雅。

代码实现：

```python
import statsmodels.api as sm

# 假设我们已经构造了一个特征（因子暴露的估计值）
# X_char: (N × T) 的因子特征矩阵
# returns: (N × T) 的收益率矩阵

T = returns.shape[1]
betas = []  # 每月的因子系数

for t in range(T):
    y_t = returns[:, t]  # 当月收益
    X_t = sm.add_constant(X_char[:, t])  # 当月特征 + 截距
    model = sm.OLS(y_t, X_t).fit()
    betas.append(model.params[1])  # 记录因子系数

# 第二步：时间序列平均
betas = np.array(betas)
beta_mean = betas.mean()
beta_se = betas.std(ddof=1) / np.sqrt(T)
t_stat = beta_mean / beta_se

print(f"因子平均月收益: {beta_mean:.4f}")
print(f"t 统计量: {t_stat:.2f}")
print(f"p 值: {2 * (1 - stats.t.cdf(abs(t_stat), df=T-1)):.4f}")
```

**怎么解读？**
- t > 2（或 p < 0.05），说明因子在统计上显著——它在样本期内有稳定的预测能力
- 系数大小 = 因子每单位暴露带来的月均超额收益——**这就是你造出来的因子收益率**

## 💼 第三步：分层回测（Quintile Analysis）

Fama-MacBeth 验证了"因子是否显著"，但作为量化策略，我们还想知道：

**用这个因子选股，实际能赚多少钱？**

分层回测的回答方式是：
1. 每个月根据因子值把所有股票分成 5 组（Quintile）
2. 做多 Q5（因子值最高的一组），做空 Q1（因子值最低的一组）
3. 看这个多空组合的累计收益

```python
# 每个月分层
def quintile_portfolios(factor, returns):
    """factor: (N, T), returns: (N, T)"""
    N, T = returns.shape
    port_returns = np.zeros((5, T))
    
    for t in range(T):
        f_t = factor[:, t]
        r_t = returns[:, t]
        
        # 按因子值分 5 组
        q = pd.qcut(f_t, 5, labels=False)
        
        for q_i in range(5):
            mask = (q == q_i)
            port_returns[q_i, t] = r_t[mask].mean()
    
    return port_returns

port_ret = quintile_portfolios(X_char, returns)

# Q5 - Q1 多空组合
long_short = port_ret[4] - port_ret[0]
cumulative_ls = (1 + long_short).cumprod()

print(f"多空组合月均收益: {long_short.mean():.4f}")
print(f"多空组合夏普比: {long_short.mean() / long_short.std() * np.sqrt(12):.2f}")
```

![分层组合累计收益](https://cdn.jsdelivr.net/gh/qumin86753649iiu/zhihu-econometric/images/025-factor-portfolio.png)
*图 1：分层组合累计收益。Q5（因子值最高组）累计收益最高，Q1（最低组）最低。灰色区域是 Q5 - Q1 多空组合的累计收益。分层单调性越好，因子的选股能力越强。*

看这张图，你一眼就能判断因子的有效性：

- **✅ 好因子**：Q5 > Q4 > Q3 > Q2 > Q1，单调分层，多空组合稳定向上
- **❌ 差因子**：分层混乱，Q5 和 Q1 的差距不显著，多空组合来回震荡

> 实战中，多空组合的**年化夏普比 > 1** 通常是一个"好因子"的粗略门槛。

## 📊 第四步：IC 分析（Information Coefficient）

分层回测告诉你"能不能赚钱"，IC 告诉你"预测的精度如何"。

IC（信息系数）= 因子值与下期收益的**截面相关系数**（通常是 Spearman 秩相关系数）。

```
ICₜ = corr(factorᵢₜ, returnᵢₜ₊₁)
```

每个月算一个 IC，得到一个 T 长的 IC 序列。

```python
from scipy.stats import spearmanr

ics = []
for t in range(T):
    ic, _ = spearmanr(X_char[:, t], returns[:, t])
    ics.append(ic)

ics = np.array(ics)
mean_ic = ics.mean()
ir = mean_ic / ics.std()  # 信息比（Information Ratio）

print(f"平均 IC: {mean_ic:.4f}")
print(f"IC 标准差: {ics.std():.4f}")
print(f"信息比 IR: {ir:.2f}")
```

IC 的解读标准：

| 平均 IC | 选股能力 | 解释 |
|--------|---------|------|
| \|IC\| ≥ 0.05 | 强 | 这个因子非常能预测收益 |
| 0.02 ≤ \|IC\| < 0.05 | 中等 | 有效但信号较弱 |
| \|IC\| < 0.02 | 弱/无效 | 可能只是噪声 |

实操中，**IR（信息比）> 0.5 的因子才值得关注**。IR = IC 均值 / IC 标准差，衡量的是 IC 的**稳定性**——一个偶尔准、经常不准的因子，IR 会很低。

## 🔬 完整代码：Fama-MacBeth 实战

把所有步骤串起来，完整的实现如下（配套代码 `025_quant_factor.py`）：

```python
# ── 第 1 步：模拟数据 ──
N, T = 500, 24
beta_true = np.random.normal(0, 1, N)
f_t = np.random.normal(0.8, 0.5, T)

returns = np.zeros((N, T))
# 构建一个可观测的"特征"——它=真实暴露+噪声
X_char = np.zeros((N, T))
for i in range(N):
    for t in range(T):
        returns[i, t] = 0.5 + beta_true[i] * f_t[t] + np.random.normal(0, 2.0)
        X_char[i, t] = beta_true[i] + np.random.normal(0, 0.5)  # 带噪声的特征

# ── 第 2 步：Fama-MacBeth ──
betas = []
for t in range(T):
    X_t = sm.add_constant(X_char[:, t])
    m = sm.OLS(returns[:, t], X_t).fit()
    betas.append(m.params[1])
betas = np.array(betas)

beta_mean = betas.mean()
beta_se = betas.std(ddof=1) / np.sqrt(T)
t_stat = beta_mean / beta_se
p_val = 2 * (1 - stats.t.cdf(abs(t_stat), df=T-1))

print(f"Fama-MacBeth 结果:")
print(f"  因子月均收益 = {beta_mean:.4f} (t = {t_stat:.2f}, p = {p_val:.4f})")

# ── 第 3 步：分层回测 ──
port_ret = np.zeros((5, T))
for t in range(T):
    q = pd.qcut(X_char[:, t], 5, labels=False)
    for qi in range(5):
        port_ret[qi, t] = returns[:, t][q == qi].mean()

ls = port_ret[4] - port_ret[0]
print(f"  多空月均收益 = {ls.mean():.4f}")
print(f"  多空夏普(年化) = {ls.mean() / ls.std() * np.sqrt(12):.2f}")

# ── 第 4 步：IC 分析 ──
ics = [spearmanr(X_char[:, t], returns[:, t])[0] for t in range(T)]
print(f"  平均 IC = {np.mean(ics):.4f}")
print(f"  IR = {np.mean(ics) / np.std(ics):.2f}")
```

## 🔗 从 Fama-MacBeth 到多因子模型

在实际的量化研究中，你不会只用单一因子。真实的多因子模型长这样：

```
Rᵢₜ = β₁×价值ᵢₜ + β₂×动量ᵢₜ + β₃×市值ᵢₜ + β₄×低波ᵢₜ + ... + εᵢₜ
```

Fama-MacBeth 的优势在处理多因子时更加明显：

1. **每月一个横截面回归**：`收益 ~ 价值 + 动量 + 市值 + 低波`
2. **得到每个因子的每月系数序列**：β̂₁ₜ, β̂₂ₜ, β̂₃ₜ, β̂₄ₜ
3. **对每个系数做 t 检验**：哪些因子显著 ≠ 0？

**多因子 Fama-MacBeth 的优点：**
- 自动处理了截面相关性（同一时期，多只股票的残差是相关的）
- 每月一个独立的截面回归，允许因子暴露随时间变化
- 系数的时间序列可以直接用来分析因子的"时变性"——比如"小盘股因子在牛市中是否更强"

**Fama-MacBeth 和分层回测的关系：**
- **Fama-MacBeth** 回答了"因子是否统计显著"——学术界标准
- **分层回测** 回答了"用因子能赚多少钱"——业界标准
- **IC 分析** 回答了"因子的预测精度和稳定性"——两种视角的桥梁

**三者互为验证。** 一个真正好的因子，三种检验都应该通过。

## 💡 延伸：为什么"选股因子" = 计量回归？

如果你回头看看整篇文章的逻辑，会发现它本质上就是计量经济学：

| 选股流程 | 对应计量概念 |
|---------|-------------|
| 提出"价值股跑赢成长股"的假设 | 提出研究假设 |
| 用回归量化因子收益 | 参数估计 |
| Fama-MacBeth 两步法 | 面板数据 + 序列相关稳健推断 |
| 分层回测 | 平均处理效应（ATE）的非参数估计 |
| IC 分析 | 相关系数检验 |
| 多空组合夏普比 | 信噪比 / t 统计量 |

**量化选股的核心，就是用计量经济学的方法，从大量数据中提取可靠的统计规律。**

第 23 篇的面板数据固定效应、第 24 篇的 Hausman 检验、第 19 篇的 DID——这些计量工具在量化研究中都有直接应用。

举两个具体的例子：

1. **行业中性化 = 固定效应**
   - 你在回归中加入行业虚拟变量，相当于在问："剔除行业差异后，这个因子还有效吗？"
   - 这就是固定效应的思想——控制了行业层面的不随时间变化的特征

2. **Barra 模型的风险因子 = 多因子回归**
   - MSCI Barra 模型（业界标准风险模型）本质上就是一个大规模的多因子截面回归
   - 因子收益率的估计方法，就是 Fama-MacBeth 的变体

所以，**学好计量经济学 = 为量化研究打下坚实的基础。**

## ✅ 小结

这一篇我们学到：

- **选股因子** = 用回归检验"某个特征能否预测未来收益"
- **Fama-MacBeth 两步法**：每月横截面回归 + 系数时间序列 t 检验——学术界金标准
- **分层回测**：将股票按因子值分成 5 组，看 Q5 是否持续跑赢 Q1——业界标准
- **IC 分析**：因子与未来收益的截面秩相关系数——衡量预测精度
- **三种方法交叉验证**：Fama-MacBeth（显著性）、分层回测（收益性）、IC（稳定性）
- **计量与量化的桥梁**：固定效应→行业中性化，Fama-MacBeth→多因子模型
